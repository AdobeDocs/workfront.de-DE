# Lookand-Feel-Updates im vierten Quartal 2024

Auf dieser Seite werden kleinere Aktualisierungen des Erscheinungsbilds verschiedener Bereiche des Adobe Workfront-Programms beschrieben, die innerhalb des Zeitrahmens des vierten Quartals 2024 vorgenommen wurden.

Eine Liste aller Änderungen, die mit der Version vom vierten Quartal 2024 verfügbar sind, finden Sie unter [Übersicht über die Version vom vierten Quartal 2024](/help/quicksilver/product-announcements/product-releases/24-q4-release-activity/24-q4-release-overview.md).

## Lookand-Feel-Aktualisierungen für Berichte

>[!NOTE]
>
>Produktionsversion für alle Kunden: Inkrementell beginnend mit Version 24.10 (17. Oktober 2024)

Wir haben die Benutzeroberfläche des Berichts an das Design anderer Bereiche von Workfront angepasst. Diese Aktualisierungen beinhalten zusätzliche Änderungen an den Reporting-Funktionen:

* Wir haben eine erweiterte Farbpalette für bedingte Formatierungsregeln in Berichten hinzugefügt. Weitere Informationen zur bedingten Formatierung finden Sie unter [Verwenden der bedingten Formatierung in Ansichten](/help/quicksilver/reports-and-dashboards/reports/reporting-elements/use-conditional-formatting-views.md).
* Wir haben eine neue Schaltfläche „Textmodus bearbeiten“ hinzugefügt, um beim Bearbeiten einer Spalte in Report Builder in den Textmodus zu wechseln. Weitere Informationen zum Bearbeiten von Berichtsansichten im Textmodus finden Sie unter [Bearbeiten einer Ansicht im Textmodus](/help/quicksilver/reports-and-dashboards/reports/text-mode/edit-text-mode-in-view.md)
* Sie können den Titel eines Berichts nicht mehr bearbeiten, indem Sie auf seinen Titel klicken. Sie können weiterhin einen Berichtstitel bearbeiten, indem Sie auf Berichtsaktionen > Bearbeiten klicken und anschließend den Titel ändern. Weitere Informationen zum Bearbeiten von Berichten finden Sie [Berichteinstellungen bearbeiten](/help/quicksilver/reports-and-dashboards/reports/creating-and-managing-reports/edit-report-settings.md).
* Sie können keine Gruppierung mehr erstellen, indem Sie im Gruppierungs-Builder auf einen Spaltennamen klicken. Sie können weiterhin eine Gruppierung basierend auf einer vorhandenen Spalte in einem Bericht erstellen, indem Sie das entsprechende Feld in der Liste Gruppieren nach: Feld auswählen. Weitere Informationen zum Erstellen von Gruppierungen finden Sie unter [Gruppierungen in Adobe Workfront erstellen](/help/quicksilver/reports-and-dashboards/reports/reporting-elements/create-groupings.md)
* Wir haben einen neuen Umschalter zur Berichtsfilterschnittstelle hinzugefügt, um beim Filtern nach einem Datumsfeld zwischen einer Kalenderdatumsauswahl und einem relativen Datums-Platzhaltertextfeld zu wechseln. Wenn Sie in der Benutzeroberfläche des Berichtsfilters außerdem eine relative Verknüpfung zum Datum als Platzhalter auswählen, werden jetzt automatisch die zugehörigen Platzhalterfelder entsprechend Ihrer Auswahl ausgefüllt. Weitere Informationen zur Verwendung relativer Datums-Platzhalter in Berichtsfiltern finden Sie unter [Verwenden von datumsbasierten Platzhaltern zum Verallgemeinern von Berichten](/help/quicksilver/reports-and-dashboards/reports/reporting-elements/use-date-based-wildcards-generalize-reports.md).
* Benutzerbasierte Platzhalter in Berichtsfiltern schlagen jetzt relevante Platzhalter für die automatische Vervollständigung basierend auf Ihrem ausgewählten Feldtyp vor. Weitere Informationen zu benutzerbasierten Platzhaltern in Berichtsfiltern finden Sie unter [Verwenden von benutzerbasierten Platzhaltern zum Verallgemeinern von Berichten](/help/quicksilver/reports-and-dashboards/reports/reporting-elements/use-user-based-wildcards-generalize-reports.md).
* Berichte mit Eingabeaufforderungen, die öffentlich freigegeben sind, werden jetzt ohne Eingabeaufforderungen angezeigt, die für alle Benutzer gelten, die über den öffentlichen Freigabe-Link darauf zugreifen. Angemeldete Benutzer, die direkt auf den Bericht zugreifen, können weiterhin Eingabeaufforderungen anwenden. Weitere Informationen zu Berichtsaufforderungen finden Sie unter [Hinzufügen einer Eingabeaufforderung zu einem Bericht](/help/quicksilver/reports-and-dashboards/reports/creating-and-managing-reports/add-prompt-report.md).
* Wir haben Änderungen an vorhandenen Berichten vorgenommen, um deren Kompatibilität mit Canvas-Dashboards zu verbessern.

_Diese Aktualisierungen werden inkrementell in der Produktion implementiert, beginnend mit der Version 24.10 am 17. Oktober 2024. Änderungen stehen zunächst nur einer kleinen Anzahl von Kundinnen und Kunden zur Verfügung und werden im Laufe der Zeit immer mehr verfügbar, bis sie ab dem 8. November 2024 für alle Kundinnen und Kunden vollständig verfügbar sind._

>[!NOTE]
>
>Produktionsversion für alle Kunden: Inkrementell beginnend mit Version 24.10 (17. Oktober 2024), volle Verfügbarkeit bis zum 8. November 2024

Die Benutzeroberfläche für Dashboards wurde entsprechend dem Design anderer Bereiche von Workfront aktualisiert. Diese Aktualisierungen beinhalten zusätzliche Änderungen an Dashboard-Funktionen:

Weitere Informationen zum Hinzufügen von Berichten und Kalendern zu einem Dashboard finden Sie unter [Dashboard bearbeiten](/help/quicksilver/reports-and-dashboards/dashboards/creating-and-managing-dashboards/edit-dashboard.md).

Allgemeine Informationen zu Dashboards finden Sie unter [Erstellen und Verwalten von Dashboards](/help/quicksilver/reports-and-dashboards/dashboards/creating-and-managing-dashboards/create-and-manage-dashboards.md).

November 2024 für alle Kundinnen und Kunden vollständig verfügbar sind._
