# Übersicht über die Versionsaktivität 2017.3

Diese Seite enthält Informationen zu den Funktionen, die in der Version 2017.3 enthalten sind. 

Die auf dieser Seite beschriebene Funktion ist derzeit in der Vorschau-Umgebung verfügbar und wird im November in der Produktionsumgebung verfügbar sein.  

>[!IMPORTANT]
>
> Die Funktionalität in der Vorschau-Umgebung kann sich vor der Veröffentlichung in der Produktionsumgebung ändern.

Die Version 2017.3 enthält Verbesserungen sowohl für Workfront-Administratoren als auch für andere Benutzer.

## Verbesserungen für Mobilgeräte {#mobile-enhancements}

>[!NOTE]
>
>Die in diesem Abschnitt beschriebene Funktion kann jetzt mit Ihrem Workfront-Produktionskonto verwendet werden. Dies ist keine Vorschaufunktion.

Im August 2017 wurde die Produktionsversion der Workfront-Mobile-App für Android und iOS um folgende Funktionen erweitert:

* Anfragen senden
* Neuer Eintrag in der Arbeitszeittabelle
* Benutzerdefiniertes Formular bearbeiten
* Genehmigungsanforderungen für Korrekturabzüge
* Push-Benachrichtigungen für abonnierte Elemente

Weitere Informationen zu diesen Änderungen finden Sie unter [Aktualisierte Mobile App für iOS und Android (Anfang August 2017)](../../../announcements/announcement-archive/updated-mobile-app.md).

Die folgenden zusätzlichen Funktionen werden ab der 8. November-Woche vor der Produktionsfreigabe verfügbar sein:

* Zugriff auf eine Liste Ihrer Projekte
* Zugriff auf eine Liste von Aufgaben und Teilaufgaben in einem Projekt
* Zugriff auf eine Problemliste in einem Projekt
* Ein neues Problem in einem Projekt protokollieren
