# Übersicht über die Release-Aktivitäten 2017.3

Auf dieser Seite finden Sie Informationen zu den Funktionen der Version 2017.3. 

Die auf dieser Seite beschriebene Funktion ist derzeit in der Vorschau-Umgebung verfügbar und wird im November in der Produktionsumgebung verfügbar sein.  

>[!IMPORTANT]
>
> Die Funktionalität in der Vorschau-Umgebung kann vor der Veröffentlichung in der Produktionsumgebung geändert werden.

Die Version 2017.3 enthält Verbesserungen für Workfront-Administratoren und andere Benutzer.

## Mobile Verbesserungen {#mobile-enhancements}

>[!NOTE]
>
Die in diesem Abschnitt beschriebene Funktion kann jetzt mit Ihrem Workfront-Produktionskonto verwendet werden. Dies ist keine Vorschaufunktion.

Die folgende Funktion wurde im August 2017 zur Produktionsversion der mobilen Workfront-App für Android und iOS hinzugefügt:

* Anfragen senden
* Datenblatt - Neuer Eintrag
* Benutzerdefinierte Formularbearbeitung
* Validierungsanfragen
* Push-Benachrichtigungen für abonnierte Elemente

Weitere Informationen zu diesen Änderungen finden Sie unter [Aktualisierte mobile App für iOS und Android (Anfang August 2017)](../../../announcements/announcement-archive/updated-mobile-app.md).

Die folgenden zusätzlichen Funktionen werden vor der Produktionsversion am 8. November verfügbar sein:

* Projektliste aufrufen
* Zugriff auf eine Liste von Aufgaben und Unteraufgaben in einem Projekt
* Auf eine Liste von Problemen in einem Projekt zugreifen
* Neues Problem in einem Projekt protokollieren
