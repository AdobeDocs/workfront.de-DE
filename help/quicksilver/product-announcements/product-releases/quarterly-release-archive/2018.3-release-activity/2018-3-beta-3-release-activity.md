# Veröffentlichungen für Version 2018.3 Beta 3

Auf dieser Seite werden alle Änderungen beschrieben, die zuletzt in der Vorschau-Umgebung mit Beta 3 Version 2018.3 verfügbar waren. Die Funktion ist ab dem 16. August 2018 in der Vorschau-Umgebung verfügbar. Sie wird im November 2018 in der Produktionsumgebung bereitgestellt.

>[!NOTE]
>
> Die auf dieser Seite beschriebene Funktion kann sich vor der Verfügbarkeit in der Produktionsumgebung ändern.

Eine Liste aller in 2018.3 vorgenommenen Änderungen finden Sie unter [Übersicht über die Versionsaktivität 2018.3](../../../../product-announcements/product-releases/quarterly-release-archive/2018.3-release-activity/2018-3-release-activity-overview.md).

* [Aktualisieren von Rollen im Benutzerprofil als Gruppenadministrator](#update-roles-in-the-user-profile-as-a-group-administrator)

* [Priorisieren Sie Ihre Arbeit zu Hause](#prioritize-your-work-in-home)
* [Eindeutige URL für die Ressourcenplaner-Benutzeransicht](#unique-url-for-the-resource-planner-user-view)
* [Profilbild in die Workfront Mobile App hochladen](#upload-profile-picture-in-the-workfront-mobile-app) 
* [Verbesserung von benutzerdefinierten Formularen in der Workfront Mobile App](#custom-form-improvement-in-the-workfront-mobile-app)
* [Zeit in der Workfront Mobile App in Querformat erfassen](#log-time-in-the-workfront-mobile-app-in-landscape-orientation)
* [Neuanordnen der Symbole in der unteren Navigationsleiste in der Workfront Mobile App](#rearrange-the-icons-in-the-bottom-navigation-bar-in-the-workfront-mobile-app)

## Der Bereich „Startseite“ ist jetzt die Standardeinstellung {#home-area-is-now-the-default}

Wir haben alle Beta-Tags im Bereich Home entfernt. Bei allen Layout-Vorlagen wird jetzt standardmäßig der Bereich Startseite angezeigt, um die Entfernung aus „Meine Arbeit“ in der Version 2019.1 vorzubereiten.

## Priorisieren Sie Ihre Arbeit zu Hause {#prioritize-your-work-in-home}

Mit der Sortieroption Meine Priorität können Sie jetzt alle Arbeitselemente in Ihrer Arbeitsliste priorisieren, die für Sie wichtig sind.

Zuvor konnten Sie nur Aufgaben und Probleme im Bereich Meine Arbeit priorisieren.

## Aktualisieren von Rollen im Benutzerprofil als Gruppenadministrator {#update-roles-in-the-user-profile-as-a-group-administrator}

Sie können jetzt die folgenden Felder für Benutzer aktualisieren, die zu einer von Ihnen verwalteten Gruppe gehören:

* Primäre Funktion
* Weitere Funktionen

Zuvor konnten nur System- und Benutzeradministratoren diese Felder aktualisieren. 

Weitere Informationen finden Sie unter [Gruppenadministratoren](../../../../administration-and-setup/manage-groups/group-roles/group-administrators.md).

## Eindeutige URL für die Ressourcenplaner-Benutzeransicht {#unique-url-for-the-resource-planner-user-view}

Sie können jetzt eine eindeutige URL für den Ressourcenplaner generieren, wenn sie in der Benutzeransicht angezeigt wird. Sie können die URL in ein Dashboard einbetten oder in eine neue Browser-Registerkarte oder ein neues Fenster einfügen, um die Benutzeransicht außerhalb der Registerkarte „Ressourcenplaner“ anzuzeigen.

Zuvor war diese Funktion nicht verfügbar.

Weitere Informationen finden Sie im Abschnitt „Eindeutige URL für die Benutzeransicht“ in [Übersicht zur Ressourcenplaner-Navigation](../../../../resource-mgmt/resource-planning/resource-planner-navigation.md).

## Hochladen eines Profilbilds in die Workfront Mobile App  {#upload-profile-picture-in-the-workfront-mobile-app}

Sie können jetzt ein Profilbild mit der Workfront Beta-App für Android hochladen.

Zuvor war diese Funktion nur auf der iOS-Plattform verfügbar.

## Verbesserungen bei benutzerdefinierten Formularen in der Workfront Mobile App {#custom-form-improvement-in-the-workfront-mobile-app}

Benutzerdefinierte Formulare werden jetzt auf der Hauptregisterkarte eines Objekts in der Workfront Beta-App für Android angezeigt.

Zuvor wurden benutzerdefinierte Formulare auf einer separaten Registerkarte angezeigt.

## Zeit in der Workfront Mobile App im Querformat erfassen {#log-time-in-the-workfront-mobile-app-in-landscape-orientation}

Sie können jetzt die Zeit in der Querformat-Ausrichtung in der Workfront Beta-App für Android protokollieren.

Zuvor konnte die Zeit nur im Hochformat protokolliert werden.

## Neuanordnen der Symbole in der unteren Navigationsleiste in der Workfront Mobile App {#rearrange-the-icons-in-the-bottom-navigation-bar-in-the-workfront-mobile-app}

Sie können jetzt die Reihenfolge anpassen, in der die Bereiche in der unteren Navigationsleiste angezeigt werden, wenn Sie die Beta-App von Workfront für Android verwenden.

Zuvor war diese Funktion nur auf der iOS-Plattform verfügbar.
