# Look-and-Feel-Aktualisierungen während des Zeitrahmens der Version 23.3

Auf dieser Seite werden kleinere Aktualisierungen des Erscheinungsbilds verschiedener Bereiche des Adobe Workfront-Programms beschrieben, die innerhalb des Zeitrahmens von Version 23.3 vorgenommen wurden. Diese Verbesserungen werden mindestens 2 Wochen nach der Veröffentlichung in der Vorschau in der Produktionsumgebung verfügbar gemacht.

Eine Liste aller Änderungen, die mit Version 23.3 verfügbar sind, finden Sie unter [23.3 - Versionsübersicht](/help/quicksilver/product-announcements/product-releases/23.3-release-activity/23-3-release-overview.md).

## Verbesserungen der Benutzeroberfläche für Anfragenlisten

>[!NOTE]
>
>Geplante Vorschauversion: 29. Juni 2023; Geplante Produktionsversion: 13. Juli 2023

Die Benutzeroberfläche der Liste „Gesendete Anfragen“ wurde verbessert.

Die Verbesserungen sind:

* Beim Hochladen der Seite wird nun eine Ladeanzeige angezeigt.
* Das Kopiersymbol ist persistent und von Text begleitet und wird deaktiviert, wenn keine Anforderungen ausgewählt sind.

Weitere Informationen finden Sie unter [Kopieren und Senden von Anfragen](/help/quicksilver/manage-work/requests/create-requests/copy-and-submit-requests.md). Die Dokumentation wird nach den Funktionsveröffentlichungen aktualisiert.

## Aktualisierter Entwurf beim Hinzufügen eines neuen Problems zu einem Projekt

>[!NOTE]
>
>Geplante Vorschau-Version: 27. Juli 2023; Geplante Produktion für die Schnellversion: Mit der Version 23.8. Geplante Produktion für die vierteljährliche Veröffentlichung: Mit Version 23.10
>
>Diese Funktion sollte ursprünglich mit Version 23.3 veröffentlicht werden und verzögerte sich später.

Wir haben das Feld „Neues Problem“ aktualisiert, das angezeigt wird, wenn ein neues Problem an ein Projekt gesendet wird. Jetzt stimmt die Benutzeroberfläche mit dem Feld Neue Anfrage überein, das angezeigt wird, wenn eine neue Anfrage an eine Anfrage-Warteschlange gesendet wird.

Weitere Informationen finden Sie unter [Probleme erstellen](../../../manage-work/issues/manage-issues/create-issues.md). Die Dokumentation wird nach den Funktionsveröffentlichungen aktualisiert.

## Aktualisierte Liste der Abrechnungssätze

>[!NOTE]
>
>Vorschau-Version: 29. Juni 2023; Geplante Produktionsversion für die Fast-Release-Kunden: Wird nach der Produktionsversion 23.3 angekündigt; Geplante Produktionsversion für alle Kunden: Mit der Version 23.4

Die Liste Projekt-Abrechnungssätze hat jetzt das modernere Erscheinungsbild, das in den meisten anderen Listen in Workfront zu finden ist.
