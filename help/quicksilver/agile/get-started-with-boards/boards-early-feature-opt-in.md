# Frühzeitiges Opt-in für Funktionen [!DNL Workfront] [!UICONTROL Boards]

Das frühe Funktions-Opt-in für [!DNL Adobe Workfront] [!UICONTROL Boards] wurde im Juli 2022 eingeführt.

[!DNL Adobe Workfront] [!UICONTROL Pinnwände] sind flexible Tools, die die Zusammenarbeit von Teams ermöglichen, indem sie Zugriff auf eine freigegebene Pinnwand bieten, die Spalten und Karten enthält.

Allgemeine Informationen zu Pinnwänden finden Sie unter [[!UICONTROL Pinnwände] - Übersicht](/help/quicksilver/agile/boards-overview.md).

## Opt-in für frühe Funktionen

### Verfügbarkeit

Frühe Funktionen für Dashboards stehen allen Unternehmen zur Verfügung. Die Beteiligung ist völlig optional.

### Opt-in

Nur [!DNL Workfront] Administratoren können sich für frühe Funktionen für Pinnwände anmelden. Wenn ein [!DNL Workfront]-Administrator frühe Funktionen aktiviert, werden alle Benutzenden in der Organisation angemeldet.

{{step1-to-boards}}

1. Klicken Sie ![Early Feature Opt](assets/early-feature-opt-in-not-enabled.png) ([!UICONTROL Early Feature Opt-in]) oben im Dashboard-Bildschirm.
1. Klicken Sie **[!UICONTROL Aktivieren]**.
1. Beginnen Sie mit der Nutzung der frühen Funktionen von Pinnwänden.

## Feedback senden

1. Klicken Sie auf einer Pinnwand auf die Schaltfläche **[!UICONTROL Feedback]**.
1. Füllen Sie das Formular aus und klicken Sie dann auf **[!UICONTROL Senden]**.

## Häufig gestellte Fragen zum frühzeitigen Opt-in für Funktionen

### Woher weiß ich, ob sich mein Unternehmen für frühe Funktionen entschieden hat?

Wenden Sie sich an Ihren [!DNL Workfront], um herauszufinden, ob er frühe Funktionen für Pinnwände aktiviert hat.

### Können frühe Funktionen nur für bestimmte Benutzer aktiviert werden?

Wenn der [!DNL Workfront]-Administrator frühe Funktionen für Pinnwände aktiviert, werden alle Benutzenden in der Organisation angemeldet.

Der [!DNL Workfront]-Administrator kann jedoch die Option Pinnwände im [!UICONTROL Hauptmenü] für einzelne Benutzer oder Benutzergruppen mit einer Layout-Vorlage ausblenden. Weitere Informationen finden Sie unter [Anpassen des [!UICONTROL Hauptmenüs] mithilfe einer Layout-Vorlage](/help/quicksilver/administration-and-setup/customize-workfront/use-layout-templates/customize-main-menu.md).

>[!NOTE]
>
>Dies hindert Benutzende nicht daran, über eine URL auf Pinnwände zuzugreifen. Dadurch wird nur verhindert, dass die Option [!DNL Boards] im [!UICONTROL Hauptmenü“ angezeigt ].

### Kann ich die frühen Funktionen deaktivieren, sobald ich Mitglied geworden bin?

Nur [!DNL Workfront] Administratoren können den Erhalt frühzeitiger Funktionen deaktivieren. Wenn ein [!DNL Workfront] die frühen Funktionen deaktiviert, werden alle Benutzenden in der Organisation abgemeldet.

{{step1-to-boards}}

1. Klicken Sie ![Early Feature Opt](assets/early-feature-opt-in-enabled.png)in aktiviert[!UICONTROL  (Early Feature Opt-in ]) oben im Dashboard-Bildschirm.
1. Klicken Sie **[!UICONTROL Deaktivieren]**.
