# Verwenden von Agile in einem Projekt

Sie können die Agile-Funktionen für Ihr Projekt nutzen, ohne die administrativen Herausforderungen zu bewältigen, die für Agile-Praktiken typisch sind (z. B. die Verwaltung des Rückstands eines Teams, die Erstellung von Iterationen usw.). Weitere Informationen finden Sie unter [Verwalten eines Projekts in der [!UICONTROL Agile]Ansicht](../../manage-work/projects/manage-projects/manage-projects-in-agile-view.md).
