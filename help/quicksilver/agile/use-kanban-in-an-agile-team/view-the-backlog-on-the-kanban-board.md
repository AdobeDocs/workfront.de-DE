# Fügen Sie dem [!UICONTROL Kanban] Pinnwand

Sie können eine [!UICONTROL Rückstand] -Spalte direkt auf Ihrem Kanban-Board. Die [!UICONTROL Rückstand] in der Spalte [!DNL Kanban] Die Pinnwand enthält die ersten 20 Elemente aus Ihrem Rückstand im [!DNL Kanban] Pinnwand. Aufgaben und Probleme müssen zu einem Projekt mit einem Status gehören, der [!UICONTROL Aktuell] im Backlog angezeigt.

## Fügen Sie dem [!UICONTROL Kanban] Pinnwand

Standardmäßig wird die [!UICONTROL Rückstand] -Spalte ist ausgeblendet. So zeigen Sie den Rückstand an:

1. Klicken Sie auf *[!UICONTROL *Hauptmenü]** Symbol ![](assets/main-menu-icon.png) in der oberen rechten Ecke von [!DNL Adobe Workfront]Klicken Sie auf **[!UICONTROL Teams]**.

1. (Optional) Klicken Sie auf die **[!UICONTROL Switch Team]** icon ![Symbol &quot;Team wechseln&quot;](assets/switch-team-icon.png), wählen Sie entweder eine neue [!UICONTROL Kanban] Team aus dem Dropdown-Menü oder suchen Sie in der Suchleiste nach einem Team.

1. Gehen Sie zur Kanban-Pinnwand, wo Sie den Rückstand anzeigen möchten.
1. Klicken **[!UICONTROL Rückstand anzeigen]**.

>[!NOTE]
>
>Wenn Sie einem Rückholelement mehrere Teams hinzufügen, wird die Aufgabe bzw. das Problem nur im Rückstand des Hauptteams angezeigt. Das Hauptteam ist das zuerst zugewiesene Team.

Weitere Informationen zum Rückstand finden Sie unter [Verwalten des agilen Rückprotokolls](../../agile/work-in-an-agile-environment/manage-the-agile-backlog.md).
