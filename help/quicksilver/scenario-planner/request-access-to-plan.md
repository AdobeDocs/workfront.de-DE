Sie können den Zugriff auf einen Plan im [!DNL Adobe Workfront Scenario Planner] wenn der Link zum Plan für Sie freigegeben wurde.

## Voraussetzungen

Bevor Sie Zugriff auf einen Plan anfordern können, finden Sie im Abschnitt [!DNL Scenario Planner]müssen Sie über Folgendes verfügen:

* Ein Link zum Plan.

>[!NOTE]
>
>Wenn Sie keine Zugriffsberechtigung für die [!DNL Scenario Planner] und Sie versuchen, über einen Link auf einen Plan zuzugreifen, können Sie keinen Zugriff auf den Plan anfordern. Stattdessen wird ein Bildschirm angezeigt, der Sie auffordert, den [!DNL Workfront] Administrator.

## Anfordern des Zugriffs auf Pläne im [!DNL Workfront Scenario Planner]

Wenn Sie noch keine Berechtigungen für einen Plan haben und von einem für Sie freigegebenen Link zu ihm navigieren, wird ein Bildschirm angezeigt, auf dem Sie darüber informiert werden, dass Sie nicht über die Berechtigung zum Anzeigen des Plans verfügen. Sie werden aufgefordert, vom Planersteller Berechtigungen anzufordern.

>[!TIP]
>
>Sie können nur Berechtigungen vom Eigentümer oder Ersteller eines Plans anfordern. Sie können keine Berechtigungen von anderen Benutzern anfordern, die auch Zugriff auf den Plan haben.

So fordern Sie Berechtigungen an:

1. Klicken Sie auf einen Link zu einem Plan.

   ![](assets/request-access-to-plan-350x277.png)

1. Im **[!UICONTROL Anfordern des Zugriffs auf]** Dropdown-Menü angeben, welche Berechtigungen Sie erhalten möchten. Wählen Sie aus den folgenden Optionen aus:

   * [!UICONTROL Anzeigen]
   * [!UICONTROL Verwalten]

   Sie können keine Berechtigung anfordern, die höher ist als Ihre Zugriffsstufe für die [!DNL Scenario Planner]. Beispielsweise können Sie keine Anforderung [!UICONTROL Verwalten] -Berechtigungen, wenn Sie Zugriff auf die Ansicht haben [!DNL Scenario Planner].

   Informationen zu den verschiedenen Berechtigungsstufen finden Sie unter [Freigeben eines Plans im [!DNL Scenario Planner]](../scenario-planner/share-a-plan.md).

   Informationen zu einem Workfront-Administrator erhalten Sie, wenn Sie den Zugriff auf [!DNL Scenario Planner], siehe [Zugriff auf gewähren [!DNL Scenario Planner]](../administration-and-setup/add-users/configure-and-grant-access/grant-access-sp.md).

1. (Optional) Geben Sie einen Kommentar oder eine Anforderung in die **[!UICONTROL Kommentarfeld verlassen]** Klicken Sie auf **[!UICONTROL Zugriff anfordern]**.

   Folgendes geschieht:

   * [!DNL Workfront] sendet eine E-Mail-Benachrichtigung an den Planinhaber, der die angeforderten Berechtigungen erteilen kann.\
      ![](assets/request-access-to-plan-email-350x156.png)

   * Nachdem der Planinhaber die angeforderten Berechtigungen erteilt hat, erhalten Sie eine E-Mail, in der Sie die Berechtigungen erhalten haben, wenn Ihr [!DNL Workfront] -Administrator hat die Objektfreigabe für die Benutzerbenachrichtigung in Ihrem System aktiviert und Sie aktivieren die [!UICONTROL Jemand teilt mir ein Objekt] E-Mail-Benachrichtigung in Ihrem Profil.

      ![](assets/access-granted-to-plan-email-350x172.png)

   * Sie können auch Berechtigungen für Pläne über die [!UICONTROL Startseite] und von [!DNL Workfront] mobile App.
   Informationen zum Aktivieren von Systembenachrichtigungen finden Sie unter [Ereignisbenachrichtigungen für alle Benutzer im System konfigurieren](../administration-and-setup/manage-workfront/emails/configure-event-notifications-for-everyone-in-the-system.md).

   Informationen zum Aktivieren von Benachrichtigungen in Ihrem Profil finden Sie unter [Benachrichtigungen: Verschiedene Informationen](../workfront-basics/using-notifications/notifications-misc-information.md).
