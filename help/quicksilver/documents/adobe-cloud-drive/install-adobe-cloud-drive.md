# Installieren von Adobe Cloud Drive

Um mit Ihren Adobe-Cloud-Speicherprojekten auf Ihrem Desktop zu arbeiten, installieren Sie Adobe Cloud Drive auf Ihrem Mac- oder Windows-Computer.

Informationen dazu, was Adobe Cloud Drive ist und was Sie damit tun können, finden Sie unter [Übersicht über Adobe Cloud Drive](/help/quicksilver/documents/adobe-cloud-drive/adobe-cloud-drive-overview.md).

## Voraussetzungen

Stellen Sie vor der Installation von Adobe Cloud Drive Folgendes sicher:

* Ihr Gerät erfüllt die im Abschnitt [Systemanforderungen](#system-requirements) dieses Artikels aufgeführten Systemanforderungen.
* Ihr Unternehmen verwendet eine Version von Workfront, die die Adobe-Cloud-Datenspeicherung unterstützt.

Wenn Ihr Unternehmen Anwendungsinstallationen für Sie verwaltet und Adobe Cloud Drive nicht zum Herunterladen verfügbar ist, wenden Sie sich an Ihren Workfront-Administrator.

## Systemanforderungen

**Betriebssysteme:**

* macOS 13.5 oder neuer (Intel und Apple-Chip)
* Windows 11 22H2 (22621.6060) oder neuer (x64)
* Windows 11 23H2 (22631.6199) oder neuer (ARM)

**Hardware:**

* Mindestens 4 GB RAM
* 500 MB freier Speicherplatz für die Anwendung
* Zusätzlicher Speicherplatz für zwischengespeicherte Dateien (variiert je nach Nutzung)

**Netzwerk:**

* Internetverbindung für Ersteinrichtung und Synchronisation
* Stabile Verbindung für optimale Leistung empfohlen

Um sicherzustellen, dass Adobe Cloud Drive die Services erreichen kann, von denen es abhängt, lassen Sie die Netzwerkendpunkte zu, die unter [Adobe-Netzwerkendpunkte](https://helpx.adobe.com/in/enterprise/kb/network-endpoints.html) aufgeführt sind.

## Herunterladen des Installationsprogramms

1. Wechseln Sie zu [Adobe-Software-Verteilung](https://experience.adobe.com/#/downloads).

1. Melden Sie sich mit Ihrer Adobe ID an.
1. Suchen Sie in der **Allgemein** Downloads nach **Adobe Cloud Drive**.
1. Wählen Sie das Installationsprogramm für Ihr Betriebssystem aus:

   * **macOS:** Adobe Cloud Drive für Mac (ARM64 oder Intel)
   * **Windows:** Adobe Cloud Drive für Windows (x64 oder ARM64)

1. Klicken Sie auf **Herunterladen**.

   >[!IMPORTANT]
   >
   >Wenn Sie das Adobe Cloud Drive-Installationsprogramm nicht sehen, verwendet Ihr Unternehmen möglicherweise keine Workfront-Version, die den Adobe-Cloud-Speicher unterstützt. Wenden Sie sich an Ihren Workfront-Administrator.

## Installieren von Adobe Cloud Drive

Verwenden Sie das Verfahren für Ihr Betriebssystem.

### Auf macOS installieren

1. Öffnen Sie die heruntergeladene `.dmg`.
1. Doppelklicken Sie auf das Symbol Installieren .
1. Folgen Sie dem Einrichtungsassistenten auf dem Bildschirm.

1. Melden Sie sich bei Aufforderung mit Ihrer Adobe ID an.

   Wenn bereits ein anderes Adobe-Desktop-Programm ausgeführt wird und Sie angemeldet sind, verwendet Adobe Cloud Drive diese aktive Sitzung automatisch.

### Installieren unter Windows

1. Führen Sie das heruntergeladene `.exe`-Installationsprogramm aus.

1. Befolgen Sie die Anweisungen im Installationsassistenten.
1. Klicken Sie **Aufforderung auf** Installieren“.
1. Starten Sie **Adobe Cloud Drive** über das Startmenü.
1. Melden Sie sich bei Aufforderung mit Ihrer Adobe ID an.

## Überprüfen der Installation

Nach der Installation wird Adobe Cloud Drive auf Ihrem Computer als bereitgestelltes Laufwerk angezeigt:

* **macOS:** In der Finder-Seitenleiste unter **Standorte** als **Adobe Cloud Drive**.
* **Windows:** Im Datei-Explorer unter **Dieser PC** als **Adobe Cloud Drive**.

Ihre Adobe-Cloud-Speicherprojekte sollten als Ordner im Laufwerk aufgeführt sein.

Informationen zum Arbeiten mit Projekten in der Festplatte finden Sie unter [Verwenden von Adobe Cloud Drive](/help/quicksilver/documents/adobe-cloud-drive/use-adobe-cloud-drive.md).
