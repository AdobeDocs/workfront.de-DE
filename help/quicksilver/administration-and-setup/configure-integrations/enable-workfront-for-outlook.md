# Aktivieren [!DNL Adobe Workfront for Outlook]

Bevor Benutzer beginnen können, [!DNL Adobe Workfront for Outlook], müssen Sie sie zunächst für Ihr System aktivieren.

Informationen zur Verwendung von [!DNL Workfront for Outlook] Nachdem sie aktiviert wurde, lesen Sie [[!DNL Adobe Workfront for Outlook]](../../workfront-integrations-and-apps/using-workfront-with-outlook/workfront-for-outlook.md).

## Aktivieren Sie die [!DNL Workfront] [!DNL Outlook] Add-in

1. Anmelden bei [!DNL Workfront] als Administrator.
1. Klicken Sie auf **[!UICONTROL Hauptmenü]** icon ![](assets/main-menu-icon.png) in der oberen rechten Ecke von [!DNL Adobe Workfront]Klicken Sie auf **[!UICONTROL Einrichtung]** ![](assets/gear-icon-settings.png).

1. Erweitern **[!UICONTROL System]** Klicken Sie auf **[!UICONTROL Voreinstellungen]**.

1. Stellen Sie sicher, dass **[!UICONTROL Lassen Sie die Benutzer die mobilen Anwendungen von Workfront und die [!DNL Workfront] [!DNL Outlook] Add-In]** ausgewählt ist.\
   Zusätzlich zur Zulassung der [!DNL Workfront] Mobile Apps verwendet werden, ermöglicht diese Einstellung auch die [!DNL Workfront] [!DNL Outlook] Zu verwendendes Add-In.

   Diese Option ist standardmäßig aktiviert.

1. Klicken Sie auf **[!UICONTROL Speichern]**.
