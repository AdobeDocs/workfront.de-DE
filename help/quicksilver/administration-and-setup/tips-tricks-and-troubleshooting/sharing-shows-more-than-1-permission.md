# Freigabe zeigt mehr als eine Berechtigung an

## Frage

Im Fenster Freigabe werden zwei verschiedene Berechtigungen für einen Benutzer angezeigt. Welche wird verwendet?

## Antwort

Benutzer haben die höchste Berechtigung, die auf dem Freigabescreen angezeigt wird. Weitere Informationen zu Berechtigungen finden Sie im Artikel [Überblick über die Freigabe von Berechtigungen für Objekte](../../workfront-basics/grant-and-request-access-to-objects/sharing-permissions-on-objects-overview.md).
