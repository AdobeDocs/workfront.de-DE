# Direkte Berichte erstellen

Sie können Benutzer als Direktberichte für einen anderen Benutzer festlegen. Auf diese Weise können Sie ein Organisationsdiagramm erstellen, in dem die Verwaltungsstruktur Ihres Unternehmens dargestellt wird. Weitere Informationen finden Sie unter [Organigramm anzeigen](../../../people-teams-and-groups/work-directly-with-others/view-the-org-chart.md).

## Direkte Berichte erstellen

1. Beginnen Sie mit der Bearbeitung eines Benutzers, wie unter [Benutzerprofil bearbeiten](../../../administration-and-setup/add-users/create-and-manage-users/edit-a-users-profile.md).
1. Im **Einrichtung** müssen Sie sicherstellen, dass ein Unternehmen im **Firma** -Feld.

   Dieses Feld darf nicht leer sein.

1. Im **Direkte Berichte** -Feld alle Benutzer angeben, die Berichte an den Benutzer richten, den Sie bearbeiten.

1. (Optional) Im **Berichte an** Geben Sie den Namen des Benutzers an, für den dieser Benutzer Berichte erstellt.
1. Klicken **Änderungen speichern**.
