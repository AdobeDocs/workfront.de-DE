# Drucken der Zielliste in Adobe Workfront-Zielen

>[!IMPORTANT]
>
>Die in diesem Artikel beschriebene Funktion wurde ab Version 23.1 von Workfront entfernt.
>
>Dieser Artikel wird auch kurz nach der Version 23.1 (Anfang 2023) entfernt. Zu diesem Zeitpunkt empfehlen wir, alle Lesezeichen entsprechend zu aktualisieren.

Sie können eine Liste von Zielen aus dem Bereich &quot;Zielliste&quot;der Adobe Workfront-Ziele drucken. Beim Drucken wird eine Datei an den Drucker gesendet oder eine PDF-Datei erzeugt, wenn das Druckziel Ihres Browsers auf Als PDF speichern eingestellt ist.

## Voraussetzungen

Sie müssen über Folgendes verfügen, bevor Sie beginnen können:

* Eine Layout-Vorlage, die den Bereich Ziele im Hauptmenü enthält.

## Zielliste drucken

>[!TIP]
>
>* Beim Drucken einer Zielgruppenliste enthält die erstellte Datei nur die auf dem Bildschirm angezeigten Informationen. Elemente, die durch Filtern einer Zielgruppenliste entfernt werden, werden nicht in der PDF-Datei angezeigt.
>* Wenn Sie die Ziele in der Liste nicht vor dem Drucken der Liste erweitern, zeigt die PDF-Datei nur Ziele ohne Ergebnisse und Aktivitäten an.
>


1. Klicken Sie auf **Hauptmenü** icon ![](assets/main-menu-icon.png) > **Ziele** in der oberen rechten Ecke.

   <!-- Add this when Shell is available to all: or (if available), click the **Main Menu** icon ![Main menu icon](../goal-management/assets/three-line-main-menu-icon.png) in the upper-left corner)
   -->

   Dadurch wird der Workfront-Zielbereich geöffnet.

   Die Zielliste wird standardmäßig angezeigt.

1. (Optional) Wenden Sie Filter auf die Zielliste an, um die für Sie relevanten Ziele anzuzeigen.

   Informationen zum Filtern von Informationen in der Zielliste finden Sie unter [Informationen in Adobe Workfront-Zielen filtern](../../workfront-goals/goal-management/filter-information-wf-goals.md).

1. Klicken **Drucken**. Das Feld &quot;Drucken&quot;wird angezeigt.
1. (Bedingt) Je nach verwendetem Browser und anderen auf Ihrem Computer installierten Anwendungen können Sie aus den folgenden Zielen auswählen:

   * Drucken auf einem Drucker
   * Drucken auf PDF
   * Speichern Sie die Zielgruppenliste auf einem lokalen oder Online-Laufwerk (z. B. Google Drive).
   * Senden der Zielgruppenliste an eine andere Anwendung (z. B. One Note)

1. (Optional) Passen Sie die Druckoptionen an.
1. Klicken **Drucken**.

   Die Liste der Ziele wird Ihrem ausgewählten Ziel bereitgestellt.
