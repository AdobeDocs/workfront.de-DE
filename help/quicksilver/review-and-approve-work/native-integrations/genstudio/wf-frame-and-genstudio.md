# Erste Schritte mit der Integration von GenStudio for Performance Marketing und Frame.io

Durch die Integration von GenStudio for Performance Marketing mit Frame.io haben Sie folgende Möglichkeiten

* Verwenden von einheitlichen Workfront-Genehmigungsvorlagen zum Definieren von Prüfungs- und Genehmigungs-Workflows

* Überprüfen und Genehmigen von GenStudio for Performance Marketing-Entwurfsinhalten im Frame.io-Viewer

* Anzeigen von Überprüfungsentscheidungen in GenStudio for Performance Marketing zur endgültigen Genehmigung und Veröffentlichung

## Integrationsanforderungen

* Workfront und GenStudio for Performance Marketing müssen in derselben Identity Management System (IMS)-Organisation bereitgestellt werden.

* Benutzende können nur zu einer Workfront-Instanz innerhalb der IMS-Organisation gehören.

* Die Workfront-Instanz muss für das einheitliche Adobe-Erlebnis aktiviert sein.

* Für die Workfront-Instanz muss Adobe Enterprise Storage aktiviert sein. Weitere Informationen finden Sie unter [Aktivieren von Adobe Enterprise Storage](/help/quicksilver/administration-and-setup/set-up-workfront/configure-system-defaults/enable-esm.md).


## Verwenden von einheitlichen Workfront-Genehmigungsvorlagen zum Definieren von Genehmigungs-Workflows

Wenn der Prozess der Inhaltsüberprüfung in Ihrem Unternehmen häufig wiederholt wird oder von denselben Personen geprüft wird, können Sie einheitliche Genehmigungsvorlagen verwenden, um Prüfungs- und Genehmigungs-Workflows zu automatisieren.

### Erstellen einer einheitlichen Genehmigungsvorlage in Workfront

Sie können einfache einstufige Vorlagen für nur einen oder zwei Prüfer erstellen oder mehrstufige Vorlagen für komplexe Prüfungen mit vielen Phasen und Abhängigkeiten erstellen.

Weitere Informationen zum Erstellen von Validierungs-Workflow-Vorlagen in Workfront finden Sie unter [Erstellen einer Validierungs-Workflow-Vorlage für Dokumente](/help/quicksilver/review-and-approve-work/document-reviews-and-approvals/manage-document-approvals/create-approval-template.md).

### Auswählen oder Ändern der Vorlage in GenStudio for Performance Marketing

Wenn ein(e) Benutzende(r) eine Überprüfung in GenStudio for Performance Marketing startet, wählt er/sie einfach die gewünschte Vorlage aus. Benutzer können jederzeit problemlos jede einheitliche Validierungs-Workflow-Vorlage ändern und Validierungsverantwortliche und Phasen hinzufügen oder entfernen.

Weitere Informationen finden Sie unter [Überprüfung und Genehmigung &#x200B;](https://experienceleague.adobe.com/de/docs/genstudio-for-performance-marketing/user-guide/approve/request-review).

## Überprüfen und Genehmigen von GenStudio for Performance Marketing-Entwurfsinhalten im Frame.io-Viewer

Entwurfsinhalte können direkt in GenStudio for Performance Marketing im Frame.io-Viewer überprüft und genehmigt werden.

Mit dem Frame.io-Viewer können Sie

* Kommentare hinterlassen
* Markup-Entwurf, um anzuzeigen, was geändert werden muss
* Entscheidung treffen

Weitere Informationen zur Verwendung des Frame.io-Viewers finden Sie unter [Kommentieren von Medien](https://help.frame.io/en/articles/9105251-commenting-on-your-media).


## Anzeigen von Überprüfungsentscheidungen in GenStudio for Performance Marketing zur endgültigen Genehmigung und Veröffentlichung

Sobald das Asset den Prüfungs- und Genehmigungsprozess durchlaufen hat, können Sie die Überprüfungsentscheidung anzeigen und den Inhalt direkt in GenStudio for Performance Marketing veröffentlichen.

Weitere Informationen finden Sie unter [Genehmigte Inhalte veröffentlichen](https://experienceleague.adobe.com/de/docs/genstudio-for-performance-marketing/user-guide/approve/publish-content).
