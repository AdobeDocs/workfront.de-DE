# Anzeigen aktiver Phasen eines Korrekturabzugs

Sie können die aktuellen aktiven Phasen in einem Korrekturabzug anzeigen, der einen automatisierten Workflow enthält.

## Zugriffsanforderungen

+++ Erweitern Sie , um die Zugriffsanforderungen für die -Funktion in diesem Artikel anzuzeigen.

Sie müssen über folgenden Zugriff verfügen, um die Schritte in diesem Artikel ausführen zu können:

## Anzeigen aktiver Phasen eines Korrekturabzugs

1. Bewegen Sie den Mauszeiger über das Dokument, das den Korrekturabzug enthält, den Sie anzeigen möchten, und wählen Sie dann **Korrekturabzugs-Workflow**, um die Korrekturabzugsschritte anzuzeigen.
