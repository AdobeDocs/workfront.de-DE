# [!DNL Adobe Workfront] mit [!DNL Anaplan]

Adobe [!DNL Workfront Fusion] ermöglicht Ihnen die Integration [!DNL Workfront] mit [!DNL Anaplan] -Konto. Sie können beispielsweise Kampagnen in [!DNL Anaplan], erstellen Sie dann eine [!DNL Workfront] mit der Kampagne verknüpftes Projekt oder Programm. Verfolgte Kosten [!DNL Workfront] kann dann wieder in hochgeladen werden. [!DNL Anaplan] , um die Kampagnenleistung zu überprüfen.

Für diese Integration müssen Sie bestimmte Objekte in beiden [!DNL Anaplan] und Workfront. Die spezifischen Objekte und Konfigurationen hängen vom jeweiligen Workflow ab. Anschließend konfigurieren Sie eine Workfront Fusion-Vorlage mit der [!DNL Anaplan] und [!DNL Workfront] Objekte und Felder. [!DNL Workfront Fusion] automatisiert dann die Integration basierend auf Ihren Konfigurationen.
