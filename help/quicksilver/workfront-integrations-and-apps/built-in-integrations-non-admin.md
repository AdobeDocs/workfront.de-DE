# Überblick über integrierte Adobe Workfront-Integrationen

<!-- Audited: 12/2023 -->

Sie können [!DNL Adobe Workfront] in Drittanbieteranwendungen sowie andere [!DNL Adobe] -Produkte integrieren. Diese Integrationen können das Dienstprogramm [!DNL Workfront] erweitern und an die Anforderungen Ihres Unternehmens anpassen. Wenn Sie beispielsweise ein Kreativprofessor in der Marketing-Branche sind, können Sie mit dem Plug-in [!DNL Adobe Workfront] für [!DNL Creative Cloud] -Anwendungen Assets direkt aus Photoshop nach Workfront exportieren.

Ihr [!DNL Workfront] -Administrator muss integrierte Integrationen aktivieren und konfigurieren, bevor Sie sie verwenden können. Wenden Sie sich an Ihren [!DNL Workfront] -Administrator, um zu erfahren, welche integrierten Integrationen in Ihrem Unternehmen verwendet werden.

## Andere [!DNL Workfront] Integrationen

Die gewünschte Integration ist möglicherweise keine integrierte Integration. [!DNL Workfront] bietet jedoch andere Möglichkeiten zur Integration in die App oder den Webdienst Ihrer Wahl. Weitere Informationen zu anderen Möglichkeiten zur Integration in [!DNL Workfront] finden Sie unter [Adobe Workfront-Integrationsmethoden](../workfront-integrations-and-apps/built-in-vs-api-vs-fusion.md).
