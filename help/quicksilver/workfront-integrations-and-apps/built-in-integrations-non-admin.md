# Übersicht über integrierte Adobe Workfront-Integrationen

<!-- Audited: 12/2023 -->

Sie können [!DNL Adobe Workfront] mit Anwendungen von Drittanbietern sowie anderen [!DNL Adobe]-Produkten integrieren. Diese Integrationen können den Nutzen von [!DNL Workfront] erweitern und an die Anforderungen Ihres Unternehmens anpassen. Wenn Sie z. B. ein Kreativschaffender in der Marketing-Branche sind, können Sie das [!DNL Adobe Workfront]-Plug-in für [!DNL Creative Cloud]-Programme verwenden, um Assets direkt aus Photoshop in Workfront zu exportieren.

Ihr [!DNL Workfront] muss integrierte Integrationen aktivieren und konfigurieren, bevor Sie sie verwenden können. Wenden Sie sich an Ihren [!DNL Workfront], um zu erfahren, welche integrierten Integrationen in Ihrer Organisation verwendet werden.

## Andere [!DNL Workfront] Integrationen

Bei der gewünschten Integration handelt es sich möglicherweise nicht um eine integrierte Integration. [!DNL Workfront] bietet jedoch andere Möglichkeiten, um mit der Mobile App oder dem Webservice Ihrer Wahl zu integrieren. Weitere Informationen zu anderen Möglichkeiten der Integration mit [!DNL Workfront] finden Sie unter [Adobe Workfront-Integrationsmethoden](../workfront-integrations-and-apps/built-in-vs-api-vs-fusion.md).
