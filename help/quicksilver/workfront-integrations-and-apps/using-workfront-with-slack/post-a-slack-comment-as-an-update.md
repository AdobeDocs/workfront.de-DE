# Post einen [!DNL Slack] -Kommentar als Update

Sie können einen [!DNL Slack] -Kommentar als Update zu einem bestimmten Projekt, einer bestimmten Aufgabe oder einem bestimmten Problem in [!DNL Adobe Workfront] hinzufügen.

## Zugriffsanforderungen

Sie müssen über Folgendes verfügen:

 </col>

  </tr> 
 </tbody>

&#42;Wenden Sie sich an Ihren [!DNL Workfront] -Administrator, um zu erfahren, welchen Plan, welchen Lizenztyp oder welchen Zugriff Sie haben.\

## Voraussetzungen

Bevor Sie einen [!DNL Slack] -Kommentar als Update posten können, müssen Sie

* Konfigurieren von [!DNL Workfront for Slack]\
   Anweisungen zum Konfigurieren von [!DNL Workfront for Slack] finden Sie unter [Konfigurieren [!DNL Adobe Workfront for Slack]](../../workfront-integrations-and-apps/using-workfront-with-slack/configure-workfront-for-slack.md).

## Post einen [!DNL Slack] -Kommentar als Update in [!DNL Workfront]

1. Bewegen Sie den Mauszeiger über einen Kommentar, den Sie in einem Kanal [!DNL Slack] hinzugefügt haben, und klicken Sie dann auf das Symbol **[!UICONTROL Mehr Aktionen]** (mit drei Punkten).

1. Klicken Sie auf **[!UICONTROL Post an update] ...`<the name of your Workfront app>`**.

   >[!NOTE]
   >
   >Möglicherweise müssen Sie auf **[!UICONTROL Weitere Nachrichtenkürzel]** klicken und nach unten scrollen, um die Option **[!UICONTROL Post und Update]** zu finden.
   >
   >
1. Geben Sie im Feld **[!UICONTROL Projekt, Aufgabe oder Problem]** den Namen eines Projekts, einer Aufgabe oder eines Problems ein, dem Sie die Aktualisierung hinzufügen möchten, und klicken Sie dann auf den Namen des Projekts, der Aufgabe oder des Problems, wenn es in der Liste angezeigt wird. Sie müssen Zugriff haben, um ein Objekt anzuzeigen, bevor es in der Liste angezeigt wird.
1. (Optional) Geben Sie in das Feld **[!UICONTROL Andere Benutzer einschließen]** den Namen eines Benutzers ein, den Sie anzeigen möchten, dass Sie das Update einbeziehen möchten, und klicken Sie dann auf den Namen, wenn er in der Liste angezeigt wird.
1. Klicken Sie auf **[!UICONTROL Senden]**.

   In [!DNL Slack] wird eine Bestätigung angezeigt, die Sie darüber informiert, dass Ihre Aktualisierung veröffentlicht wurde.

1. (Optional) Klicken Sie auf den Namen des Projekts, der Aufgabe oder des Problems, für das Sie die Aktualisierung veröffentlicht haben, um in [!DNL Workfront] darauf zuzugreifen.

   Wenn ein Kommentar aus [!DNL Slack] stammt, zeigt der Kommentar die Meldung &quot;[!UICONTROL Von [!DNL Slack]] gepostet&quot;in [!DNL Workfront] an.
