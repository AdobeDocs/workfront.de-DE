# Erstellen von Testversandgruppen mit [!DNL Workfront Proof]

>[!IMPORTANT]
>
>Dieser Artikel bezieht sich auf die Funktionalität im eigenständigen Produkt [!DNL Workfront Proof]. Informationen zum Testen in [!DNL Adobe Workfront], siehe [Testversand](../../../review-and-approve-work/proofing/proofing.md).

Für die Verwendung dieser Funktion ist ein Select- oder Premium-Plan erforderlich. Weitere Informationen zu den verschiedenen verfügbaren Plänen finden Sie unter [[!DNL Workfront] Pläne](https://www.workfront.com/plans).

Sie können Benutzergruppen erstellen, die Sie als Prüfer für mehrere Testsendungen hinzufügen können. Dies ist nützlich, wenn Sie regelmäßig Inhalte an dieselben Benutzer senden.

1. Klicken Sie im Dashboard auf den Abwärtspfeil neben **[!UICONTROL Neu]** Klicken Sie auf **[!UICONTROL Neue Gruppe]**.

1. Im **[!UICONTROL Neue Gruppe]** Geben Sie Details für die Gruppe ein (Gruppennamen können bis zu 128 Zeichen lang sein).

   Auswählen **[!UICONTROL Privat]** macht die Gruppe nur für Sie und die anderen Mitglieder der Gruppe sichtbar.

1. Im **[!UICONTROL Personen]** hinzufügen oder entfernen Sie Personen aus Ihrer Gruppe, indem Sie bei Bedarf deren Rolle und E-Mail-Benachrichtigungsoptionen angeben.

   Informationen zum Festlegen von Rollen finden Sie unter [Verwalten von Proof-Rollen in [!DNL Workfront Proof]](../../../workfront-proof/wp-work-proofsfiles/share-proofs-and-files/manage-proof-roles.md).

   Informationen zu den E-Mail-Warnungsoptionen finden Sie unter [E-Mail-Benachrichtigungseinstellungen konfigurieren in [!DNL Workfront Proof]](../../../workfront-proof/wp-emailsntfctns/email-alerts/config-email-notification-settings-wp.md).

1. Klicken Sie auf **[!UICONTROL Speichern]**.
