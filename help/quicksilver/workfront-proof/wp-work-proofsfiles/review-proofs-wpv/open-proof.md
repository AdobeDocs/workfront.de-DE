# Öffnen eines Testversands in [!DNL Workfront Proof]

>[!IMPORTANT]
>
>Dieser Artikel bezieht sich auf die Funktionalität im eigenständigen Produkt [!DNL Workfront Proof]. Informationen zum Testen in [!DNL Adobe Workfront], siehe [Testversand](../../../review-and-approve-work/proofing/proofing.md).

1. Klicken Sie auf die Nach-unten-Pfeilschaltfläche neben **[!UICONTROL Ansichten]** in der Seitenleiste.\
   ![Down_arrow_next_to_Views.png](assets/down-arrow-next-to-views-193x371.png)

1. Auswählen **[!UICONTROL Alle Elemente]** im angezeigten Menü.
1. Klicken Sie auf **[!UICONTROL Gehe zu Testversand]** für den Testversand, den Sie anzeigen möchten.\
   ![Go_to_Proof_blue_icon.png](assets/go-to-proof-blue-icon.png)\
   Ihr standardmäßiger Testversand-Viewer wird in Ihrem Browser auf einer neuen Registerkarte gestartet und der Fokus wechselt zu dieser Registerkarte. Sie können mehrere Testsendungen gleichzeitig öffnen lassen, jede auf einem eigenen Tab.

1. Fahren Sie mit einem der folgenden Artikel fort, je nach verwendetem Testversand-Viewer.

   * Informationen zum Testen im Web Proofing-Viewer finden Sie unter [Überprüfen von Testsendungen im Web Proofing-Viewer.](https://support.workfront.com/hc/en-us/sections/115000275214)
   * Informationen zum Testen im Desktop Proofing Viewer finden Sie unter [Überprüfen von Testsendungen im Viewer für die Desktop-Testversion.](https://support.workfront.com/hc/en-us/search/click?data=BAh7CjoHaWRsKwjm7%2BTRUwA6CXR5cGVJIgxhcnRpY2xlBjoGRVQ6CHVybEkiVC9oYy9lbi11cy9hcnRpY2xlcy8zNjAwMDM3MjczMzQtUmV2aWV3aW5nLVByb29mcy1pbi10aGUtRGVza3RvcC1Qcm9vZmluZy1WaWV3ZXIGOwdUOg5zZWFyY2hfaWRJIik0NDIyMjdkZi0zYTA4LTQ2YjItYTdkMy1kYzM1YjhlN2U4MjUGOwdGOglyYW5raQc%3D--2056c434cf6f4f97ca87532493ebfeb67ca07b63)
   Weitere Informationen zu den Testversand-Viewern finden Sie unter [Unterschiede zwischen dem Web Proofing Viewer und dem Desktop Proofing Viewer - Übersicht](../../../review-and-approve-work/proofing/proofing-overview/understand-differences-between-web-viewer.md).
