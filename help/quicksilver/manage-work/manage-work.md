# Verwalten von Arbeitsinformationen

Sie können Adobe Workfront zur Verwaltung Ihrer Arbeit verwenden.

Sie können zunächst bestimmen, welche Arbeitsobjekte Sie zum Definieren Ihrer Arbeit verwenden, abhängig von deren Komplexität und den Ressourcen oder dem Budget, das Sie dafür zuteilen.

In diesem Abschnitt wird beschrieben, wie Sie Ihre Arbeit in größeren Arbeitselementen wie Projekten, Portfolios oder Programmen organisieren und wie Sie größere Arbeitsmengen in besser verwaltbare Arbeitselemente wie Aufgaben und Probleme unterteilen können.

Darüber hinaus können Sie das Gantt-Diagramm verwenden, um eine Timeline Ihrer Arbeit zu definieren oder Ihre Arbeit an andere zu delegieren, wenn Sie selbst nicht verfügbar sind, um sie zu erledigen.
