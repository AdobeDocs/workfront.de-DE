## Suchen Sie das [!UICONTROL Gantt-Diagramm].

Sie können die Aufgabenliste Gantt-Diagramm und die Projektliste [!UICONTROL Gantt-Diagramm] aus mehreren Bereichen in Workfront finden. Weitere Informationen finden Sie unter [Erste Schritte mit dem [!UICONTROL Gantt-Diagramm]](../../../manage-work/gantt-chart/use-the-gantt-chart/get-started-with-gantt.md) .

## Anzeigen von Aufgaben auf dem [!UICONTROL kritischen Pfad]

In der Projektliste [!UICONTROL Gantt-Diagramm] werden Aufgaben, die sich nicht auf dem [!UICONTROL Kritischen Pfad] befinden, als hellblaue horizontale Linien angezeigt. Aufgaben, die sich auf dem [!UICONTROL kritischen Pfad] eines Projekts befinden, werden als rote horizontale Linien angezeigt.

Weitere Informationen zu Aufgaben auf dem [!UICONTROL kritischen Pfad] finden Sie unter [Überblick über das Projekt [!UICONTROL Kritischer Pfad]](../../../manage-work/tasks/manage-tasks/critical-path.md).

## Aufgabeninformationen in der Projektliste anzeigen [!UICONTROL Gantt-Diagramm]

Sie können die Aufgabeninformationen für ein Projekt direkt in der Projektliste anzeigen. Aufgaben werden unter dem Namen jedes Projekts aufgeführt.

>[!NOTE]
>
>Sie können Aufgaben nicht über die Projektliste [!UICONTROL Gantt-Diagramm] bearbeiten.

Sie können die Aufgabeninformationen für ein Projekt direkt aus einer Liste von Projekten in den folgenden Bereichen anzeigen:

* Im Bereich [!UICONTROL Projekte]
* Innerhalb eines Portfolios
* Innerhalb eines Programms

So zeigen Sie Aufgaben in einem Projekt aus einer Projektliste an:

1. Zu einem der oben genannten Bereiche gehen.

   Klicken Sie beispielsweise im Hauptmenü [!UICONTROL 1} auf **[!UICONTROL Projekte]**.]

   Eine Liste der Projekte wird angezeigt.

1. Klicken Sie oben rechts im Bildschirm auf das Symbol **[!UICONTROL Gantt-Diagramm]** ![](assets/gantt-icon-nwe.png) .

   <!--
   <p data-mc-conditions="QuicksilverOrClassic.Draft mode">(NOTE: images conditioned for classic and nwe) <br></p>
   -->

1. Klicken Sie auf das Symbol **[!UICONTROL Aufgabenliste anzeigen]** .

1. Klicken Sie links in der Projektliste auf den Dropdown-Pfeil neben dem Namen des Projekts, um alle Aufgaben unter diesem Projekt anzuzeigen.\
   Dadurch werden Aufgabeninformationen im [!UICONTROL Gantt-Diagramm] angezeigt.\
   ![Show_task_list_enabled_project_expand.png](assets/show-task-list-enabled-project-expanded-350x78.png)

1. (Optional) Klicken Sie oben rechts auf das Symbol **[!UICONTROL Drucken]** , um das [!UICONTROL Gantt-Diagramm] zu exportieren.

   >[!NOTE]
   >
   >Die Projektliste [!UICONTROL Gantt-Diagramm] exportiert nur Projekte. Aufgabeninformationen sind nicht enthalten.

## Ändern Sie den Zeitraum, für den Informationen im [!UICONTROL Gantt-Diagramm] angezeigt werden.

Sie können den im [!UICONTROL Gantt-Diagramm] angezeigten Zeitraum anpassen, um Informationen granular anzuzeigen, oder Sie können schnell zu einer Tages-, Wochen-, Monats-, Quartals- oder Jahresansicht navigieren:

* [Zeitraum auf einer granularen Ebene ändern](#change-the-time-period-on-a-granular-level)
* [Anzeigen von Informationen nach Tag, Woche, Monat, Quartal oder Jahr](#view-information-by-day-week-month-quarter-or-year)

### Zeitraum auf einer granularen Ebene ändern {#change-the-time-period-on-a-granular-level}

1. Bewegen Sie den Mauszeiger über die Timeline des [!UICONTROL Gantt-Diagramms] und ziehen Sie dann den Zoom-Indikator von links nach rechts, um die Timeline zu erweitern oder zu verkleinern.\
   ![zoom_tool_in_gantt.png](assets/zoom-tool-in-gantt-350x180.png)

### Anzeigen von Informationen nach Tag, Woche, Monat, Quartal oder Jahr {#view-information-by-day-week-month-quarter-or-year}

1. Klicken Sie im [!UICONTROL Gantt-Diagramm] auf das Dropdown-Menü für den Zeitrahmen.

   ![](assets/timeline-options.png)

1. Wählen Sie einen Zeitraum aus den folgenden verfügbaren Optionen aus:

   * **[!UICONTROL Alle anpassen]**: Diese Option zeigt die Zeitlinie des gesamten Projekts an.
   * **[!UICONTROL Alle Projekte]**: Diese Option ist nur im Gantt-Diagramm der Projektliste verfügbar.
   * **[!UICONTROL Year]**
   * **[!UICONTROL Quartal]**
   * **[!UICONTROL Monat]**
   * **[!UICONTROL Woche]**
   * **[!UICONTROL Tag]**

1. (Optional) Wählen Sie einen detaillierteren Zeitrahmen aus, z. B. [!UICONTROL Woche] oder [!UICONTROL Tag], klicken Sie auf die horizontale Bildlaufleiste am unteren Rand des [!UICONTROL Gantt-Diagramms] und ziehen Sie sie nach links, um sich in der Zeitleiste des Projekts nach rechts zu bewegen.\
   Ein Timeline-Schnappschuss des [!UICONTROL Gantt] wird angezeigt, um das gesamte Projekt anzuzeigen.

   >[!TIP]
   >
   >Der Timeline-Schnappschuss wird erst angezeigt, wenn Sie auf die horizontale Bildlaufleiste klicken.

   ![gestreckchy_gantt_minimap_with_entwurfs__1_.png](assets/stretchy-gantt-minimap-with-outline--1--350x140.png)

1. (Optional) Klicken Sie auf eine beliebige Stelle innerhalb des Zeitleisten-Snapshots, um zu einem bestimmten Punkt im Projektlebenszyklus zu navigieren.\
   Oder\
   Ziehen Sie die Griffe des Snapshot-Viewers, um einen bestimmten Zeitlinienbereich auszuwählen und ihn im Hauptbereich [!UICONTROL Gantt] anzuzeigen.

## Filter, Ansichten und Gruppierungen verwenden

Das [!UICONTROL Gantt-Diagramm] ist eine visuelle Darstellung der Informationen, die derzeit in der Aufgabenliste angezeigt werden. Sie können Filter, Ansichten und Gruppierungen auf die Objekte anwenden, die in den beiden Diagrammen [!UICONTROL Gantt] aufgeführt sind.

>[!CAUTION]
>
>Sie können keine Filter, Ansichten und Gruppierungen anwenden, wenn Sie [!UICONTROL Manuell] auswählen, um [!UICONTROL Timeline-Planung] zu speichern und Änderungen an Ihrer Aufgabenliste zu speichern. Informationen zum Speichern von Änderungen an Aufgaben in einer Liste finden Sie unter [Aufgaben in einer Liste bearbeiten](../../../manage-work/tasks/manage-tasks/edit-tasks-in-a-list.md).

Filter und Gruppierungen, die Sie auf die Liste anwenden, werden sowohl in der Projektliste als auch in der Aufgabenliste [!UICONTROL Gantt-Diagramm]s angezeigt und sind auch beim Exportieren der Gantt-Diagramme enthalten:

* Filter\
   Sie können einen Filter auf die Liste anwenden, um die im [!UICONTROL Gantt-Diagramm] angezeigten Informationen zu steuern.\
   Informationen zum Anwenden eines Filters finden Sie unter [Filterübersicht](../../../reports-and-dashboards/reports/reporting-elements/filters-overview.md).

* Gruppierungen\
   Gruppierungen, die Sie auf die Liste anwenden, werden im [!UICONTROL Gantt-Diagramm] angezeigt.\
   Informationen zum Anwenden einer Gruppierung finden Sie unter [Gruppierungsübersicht in Adobe Workfront](../../../reports-and-dashboards/reports/reporting-elements/groupings-overview.md).

Ansichten werden nicht im [!UICONTROL Gantt-Diagramm] angezeigt. Wenn Sie jedoch das [!UICONTROL Gantt-Diagramm] exportieren (wie unter [Exportieren des [!UICONTROL Gantt-Diagramms] in PDF](../../../manage-work/gantt-chart/use-the-gantt-chart/export-gantt-chart-to-pdf.md) beschrieben), wird die Aufgabenliste zusätzlich zum [!UICONTROL Gantt-Diagramm] exportiert, wobei die aktuelle Ansicht auf die Liste angewendet wird.

## Anzeigeoptionen konfigurieren

Sie können festlegen, welcher Informationstyp in beiden [!UICONTROL Gantt-Diagrammen] angezeigt wird. Weitere Informationen finden Sie unter [Konfigurieren der Anzeige von Informationen im [!UICONTROL Gantt-Diagramm]](../../../manage-work/gantt-chart/use-the-gantt-chart/configure-info-on-gantt-chart.md).
