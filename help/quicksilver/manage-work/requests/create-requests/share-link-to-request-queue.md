## Freigeben eines Links zu einer Anfrage-Warteschlange

1. Senden Sie eine Anfrage wie in [Erstellen und Senden von Adobe Workfront-Anfragen](../../../manage-work/requests/create-requests/create-submit-requests.md) beschrieben.
1. Führen Sie einen der folgenden Schritte aus:

   * Wählen Sie eine Anfrage-Warteschlange im Feld Anfragetyp aus
   * Anfrage-Warteschlange und bis zu 10 Themengruppen auswählen
   * Anfrage-Warteschlange, Themengruppe und Warteschlangenthema auswählen

1. Bevor Sie die Anfrage senden, klicken **oben rechts** der Seite auf „Pfad freigeben“.

   Dadurch wird ein Link zur Anfragewarteschlange, Themengruppe oder zum Warteschlangenthema, das Sie im vorherigen Schritt ausgewählt haben, in die Zwischenablage kopiert.

   <!--
   <p data-mc-conditions="QuicksilverOrClassic.Draft mode">(NOTE: does this step stay accurate?) </p>

1. Führen Sie einen der folgenden Schritte aus:

   * Freigeben des kopierten Links für andere Benutzer
   * Einbetten in ein Dashboard als externe Seite

     Informationen zum Einbetten einer Anfrage-Warteschlange in ein Dashboard finden Sie unter [Anfrage-Warteschlange in ein Dashboard einbetten](../../../reports-and-dashboards/dashboards/creating-and-managing-dashboards/embed-request-queue-dashboard.md).

   Über den freigegebenen Link können andere Benutzende direkt zur Anfrage-Warteschlange, zur Themengruppe oder zum ausgewählten Warteschlangenthema navigieren. Dies ist auch dann möglich, wenn die Anfrage nicht übermittelt und als Entwurf gespeichert wird.
