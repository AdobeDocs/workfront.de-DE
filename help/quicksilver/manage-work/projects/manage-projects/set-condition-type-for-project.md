# Bedingungstyp eines Projekts festlegen

## Bedingungstyp für ein Projekt festlegen

1. Wechseln Sie zu dem Projekt, für das Sie den Bedingungstyp aktualisieren möchten.
1. Führen Sie einen der folgenden Schritte aus: 

   * Klicken Sie auf **Mehr** Menü ![](assets/qs-more-menu.png) rechts neben dem Projektnamen klicken Sie auf **Bearbeiten**.
   * Klicken **Projektdetails** im linken Bereich.

   ![](assets/update-condition-type-nwe-350x108.png)

1. Im **Bedingungstyp** auswählen, wählen Sie eine der folgenden Optionen:

   * **Manuell:** Der Projekteigentümer legt die Bedingung für das Projekt manuell fest.

      In diesem Fall kann der Projekteigentümer die Bedingung des Projekts im Projektheader oder im Abschnitt Projektdetails aktualisieren.

   * **Fortschrittsstatus:** Workfront legt die Bedingung auf Grundlage des Fortschrittsstatus des Projekts fest. Informationen zur Berechnung des Fortschrittsstatus finden Sie unter [Übersicht über den Projektstatus](../../../manage-work/projects/planning-a-project/project-progress-status.md).

1. Klicken **Speichern** wenn Sie den Bedingungstyp im Feld &quot;Projekt bearbeiten&quot;ändern.

   Klicken **Änderungen speichern** wenn Sie den Bedingungstyp im Abschnitt Projektdetails ändern.
