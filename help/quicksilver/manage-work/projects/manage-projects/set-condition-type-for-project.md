# Bedingungstyp eines Projekts festlegen

## Bedingungstyp für ein Projekt festlegen

1. Wechseln Sie zu dem Projekt, für das Sie den Bedingungstyp aktualisieren möchten.
1. Führen Sie einen der folgenden Schritte aus: 

   * Klicken Sie auf das Menü **Mehr** rechts neben dem Projektnamen und dann auf **Bearbeiten**.![](assets/qs-more-menu.png)
   * Klicken Sie im linken Bereich auf **Projektdetails** .

   ![](assets/update-condition-type-nwe-350x108.png)

1. Wählen Sie im Feld **Bedingungstyp** eine der folgenden Optionen aus:

   * **Manuell:** Der Projektinhaber legt die Bedingung für das Projekt manuell fest.

     In diesem Fall kann der Projekteigentümer die Bedingung des Projekts im Projektheader oder im Abschnitt Projektdetails aktualisieren.

   * **Fortschrittsstatus:** Workfront legt die Bedingung auf Grundlage des Fortschrittsstatus des Projekts fest. Informationen zur Berechnung des Fortschrittsstatus finden Sie unter [Übersicht über den Projektfortschritt](../../../manage-work/projects/planning-a-project/project-progress-status.md).

1. Klicken Sie auf **Speichern** , wenn Sie den Bedingungstyp im Feld &quot;Projekt bearbeiten&quot;ändern.

   Klicken Sie auf **Änderungen speichern** , wenn Sie den Bedingungstyp im Abschnitt &quot;Projektdetails&quot;ändern.
