# Übersicht über In-App-Benachrichtigungen

In diesem Artikel werden die In-App-Benachrichtigungen aufgelistet, die Sie erhalten können. In-App-Benachrichtigungen informieren Sie über zwei Arten von Informationen: Benachrichtigungen und Benachrichtigungen zu Arbeitselementen. Sie sind sowohl in der Webanwendung als auch in der Mobile App verfügbar.

Anweisungen zum Anzeigen und Verwalten dieser Benachrichtigungen finden Sie unter [Anzeigen und Verwalten von In-App-Benachrichtigungen](../../workfront-basics/using-notifications/view-and-manage-in-app-notifications.md).

* [[!UICONTROL Mitteilung] Benachrichtigungen](#announcement-notifications)
* [[!UICONTROL Arbeitselement] Benachrichtigungen](#work-item-notifications)

## [!UICONTROL Mitteilung] Benachrichtigungen

Mitteilungen bieten zwei Möglichkeiten:

* [!DNL Workfront] sendet eine Nachricht an die [!UICONTROL Mitteilung] Seite der [!DNL Workfront] Administratoren
* A [!DNL Workfront] -Administrator sendet eine Nachricht von der [!UICONTROL Mitteilung] page

In beiden Fällen, wenn der Administrator oder Benutzer auf das nummerierte Symbol klickt ![](assets/notifications-icon-jewel.jpg) in der oberen rechten Ecke von [!DNL Workfront] um die Benachrichtigungsliste zu öffnen, wird die Betreffzeile der Nachricht mit einem Megaphon-Symbol angezeigt ![](assets/announcement.png) und das Wort &quot;[!UICONTROL Mitteilung].&quot;

![[!UICONTROL Mitteilung] Benachrichtigung](assets/notification-list-announcement-350x271.png)

Informationen zum [!UICONTROL Mitteilungen] Seite, siehe [Anzeigen und Verwalten von In-App-Benachrichtigungen](../../workfront-basics/using-notifications/view-and-manage-in-app-notifications.md).

## [!UICONTROL Arbeitselement] Benachrichtigungen

Wenn Sie auf das nummerierte Symbol klicken ![](assets/notifications-icon-jewel.jpg) in der oberen rechten Ecke von [!DNL Workfront] um Ihre Benachrichtigungsliste zu öffnen, wird die Betreffzeile der Nachricht zum Arbeitselement mit einem Symbol und einer kurzen Beschreibung angezeigt, um zu ermitteln, worum es geht. Die Symbole und Beschreibungsarten sind unten aufgeführt.

![Arbeitselementbenachrichtigungen](assets/notification-list-work-items-350x247.png)

>[!NOTE]
>
>In-App-Benachrichtigungen für Arbeitsaktivitäten sind nicht an E-Mail-Benachrichtigungen in [!DNL Workfront]. Weitere Informationen finden Sie unter [[!DNL Adobe Workfront] Benachrichtigungen](../../workfront-basics/using-notifications/wf-notifications.md).
