# Linke Navigation in [!DNL Adobe Workfront]

Die meisten Bereiche und Objekte in WF verwenden ein einfaches Navigationsfenster auf der linken Bildschirmseite. Die Navigation im linken Bereich bietet folgende Vorteile:

* Dies ermöglicht eine bessere Verwaltung des Bildschirmbereichs.
* Ihre [!DNL Workfront] -Administrator kann jeden Bereich im linken Bereich ausblenden, mit Ausnahme der Objektdetails mithilfe von Layoutvorlagen.

   Informationen zur Verwendung von Layoutvorlagen finden Sie im Artikel [Erstellen und Verwalten von Layoutvorlagen](../../administration-and-setup/customize-workfront/use-layout-templates/create-and-manage-layout-templates.md).

* Sie können die Reihenfolge der Abschnitte im linken Navigationsbereich einfach ändern, ohne zusätzliche Bildschirme zu öffnen, indem Sie sie in die gewünschte Reihenfolge ziehen und dort ablegen.

   Weitere Informationen finden Sie im folgenden Abschnitt in diesem Artikel: [Verwenden des linken Navigationsbereichs](#use-the-left-navigation-panel).

* Sie können das linke Bedienfeld anpassen, indem Sie ein Dashboard hinzufügen.

## Standardabschnitte im linken Navigationsbereich

Weitere Informationen zu Objekten oder Bereichen finden Sie im linken Bereich für verschiedene Objekte oder Bereiche in Adobe Workfront.

Je nach Objekt oder Gebiet, zu dem Sie navigiert sind, sind die im linken Bereich angezeigten Abschnitte unterschiedlich.

Der linke Bereich ist für die folgenden Objekte verfügbar:

* Projekt
* Aufgabe
* Anfrage
* Portfolio
* Programm
* Vorlage
* Vorlagenaufgabe
* Wiederholung
* Benutzer
* Team
* Gruppe
* Ziel

>[!IMPORTANT]
>
>Wenn im linken Bereich standardmäßig nur ein Abschnitt angezeigt wird, z. B. **[!UICONTROL Alle Projekte]** im **[!UICONTROL Projekte]** area—a [!DNL Workfront] Der Administrator muss diesem Bereich beim Erstellen einer Layout-Vorlage mindestens einen benutzerdefinierten Abschnitt hinzufügen und Sie dieser Vorlage zuweisen, bevor das linke Bedienfeld in diesem Bereich angezeigt wird.\
>Informationen zum [!DNL Workfront] Administrator passt den linken Bereich in einer Layoutvorlage an, siehe [Anpassen des linken Bedienfelds mithilfe einer Layoutvorlage](../../administration-and-setup/customize-workfront/use-layout-templates/customize-left-panel.md).

## Verwenden des linken Navigationsbereichs

Das Suchen und Bearbeiten von Informationen im linken Bereich ist für diese Objekte ähnlich. Die im linken Bereich verfügbaren Optionen variieren je nach dem Objekt, auf das Sie zugreifen.

Informationen dazu, welche Abschnitte für bestimmte Bereiche verfügbar sind, finden Sie unter [[!UICONTROL Standardabschnitte] im linken Navigationsbereich](#default-sections-in-the-left-navigation-panel).

1. Klicken Sie auf **[!UICONTROL Hauptmenü]** icon ![](assets/main-menu-icon-16x12.png)und klicken Sie dann auf den Namen eines Objekts, um darauf zuzugreifen.

   Die Objektseite wird angezeigt.

1. (Bedingt) Wenn Sie einen Bereich ausgewählt haben, der eine Liste von Objekten enthält, z. B. **[!UICONTROL Projekte]** oder **[!UICONTROL Portfolios]**—Sie müssen Folgendes tun, um auf die Navigation im linken Bedienfeld zuzugreifen:

   1. Klicken Sie auf ein Objekt in der Liste.
   1. (Optional) In der linken Bereichsnavigation können Sie ein anderes Objekt auswählen, z. B. **[!UICONTROL Aufgaben]** oder **[!UICONTROL Probleme]** in einem Projekt - im linken Bereich, um auf eine Liste von Objekten zuzugreifen, und klicken Sie dann auf den Namen des spezifischen Objekts.\

      Die Objektseite wird angezeigt.

1. Klicken Sie auf der Objektseite auf einen der Abschnitte im linken Bereich, um die Informationen in diesem Abschnitt anzuzeigen oder zu bearbeiten.

   Sie müssen über die Berechtigungen zum Bearbeiten des Objekts verfügen, bevor Sie die Informationen bearbeiten können.

   ![Linke Bedienfeldabschnitte](assets/NWE-left-navigation-panel.png)

1. (Optional) Klicken Sie auf **[!UICONTROL Mehr anzeigen]** unten im linken Bereich, um weitere Abschnitte anzuzeigen.

   >[!NOTE]
   >
   >Diese Option ist nicht verfügbar, wenn standardmäßig alle Abschnitte im linken Bereich angezeigt werden.

1. (Optional) Um einen Schnelllink zum Dashboard hinzuzufügen, klicken Sie auf **[!UICONTROL Dashboard hinzufügen]** Gehen Sie unten im linken Bereich wie folgt vor:

   1. Geben Sie einen Namen für das Dashboard im **[!UICONTROL Name des Schnelllinks]** -Feld.
   1. Geben Sie den Namen eines vorhandenen Dashboards im **[!UICONTROL Dashboard auswählen]** und dann auf das Dashboard klicken, wenn es in der Liste angezeigt wird.

      >[!TIP]
      >
      >Sie müssen das Dashboard erstellen, bevor es in der Liste angezeigt wird.

   1. Klicken **[!UICONTROL Hinzufügen]**.

      Der Dashboard-Link wird unten im linken Bereich angezeigt.

1. (Optional) Um die Reihenfolge eines Bereichs im linken Bereich zu ändern, klicken Sie auf die Schaltfläche **[!UICONTROL Ziehen]** icon ![](assets/drag-icon.png) rechts neben einen Abschnitt und ziehen Sie ihn nach oben oder unten.

   ![Abschnitt ziehen](assets/drag-section-right.png)

1. (Optional) Um das linke Bedienfeld zu schließen, klicken Sie auf das **[!UICONTROL Reduzieren]** icon ![](assets/left-panel-collapse-icon.png).

   >[!NOTE]
   >
   >Wenn Sie das linke Bedienfeld reduzieren, [!DNL Workfront] behält Ihre Voreinstellung bei, selbst wenn Sie zu einem anderen navigieren [!DNL Workfront] -Objekt. Um das linke Bedienfeld erneut zu erweitern, klicken Sie auf das **[!UICONTROL Erweitern]** icon ![](assets/left-panel-expand-icon.png).
