# Linke Navigation in [!DNL Adobe Workfront]

Die meisten Bereiche und Objekte in WF verwenden ein einfaches Navigationsfenster auf der linken Bildschirmseite. Die Navigation im linken Bereich bietet folgende Vorteile:

* Dies ermöglicht eine bessere Verwaltung des Bildschirmbereichs.
* Ihr [!DNL Workfront] -Administrator kann jeden Bereich im linken Bereich ausblenden, mit Ausnahme der Objektdetails mithilfe von Layoutvorlagen.

  Informationen zur Verwendung von Layoutvorlagen finden Sie im Artikel [Erstellen und Verwalten von Layoutvorlagen](../../administration-and-setup/customize-workfront/use-layout-templates/create-and-manage-layout-templates.md).

* Sie können die Reihenfolge der Abschnitte im linken Navigationsbereich einfach ändern, ohne zusätzliche Bildschirme zu öffnen, indem Sie sie in die gewünschte Reihenfolge ziehen und dort ablegen.

  Weitere Informationen finden Sie im folgenden Abschnitt in diesem Artikel: [Verwenden des linken Navigationsbereichs](#use-the-left-navigation-panel).

* Sie können das linke Bedienfeld anpassen, indem Sie ein Dashboard hinzufügen.

## Standardabschnitte im linken Navigationsbereich

Weitere Informationen zu Objekten oder Bereichen finden Sie im linken Bereich für verschiedene Objekte oder Bereiche in Adobe Workfront.

Je nach Objekt oder Gebiet, zu dem Sie navigiert sind, sind die im linken Bereich angezeigten Abschnitte unterschiedlich.

Der linke Bereich ist für die folgenden Objekte verfügbar:

* Projekt
* Aufgabe
* Problem
* Portfolio
* Programm
* Vorlage
* Vorlagenaufgabe
* Wiederholung
* Benutzerin oder Benutzer
* Team
* Gruppe
* Ziel

>[!IMPORTANT]
>
>Wenn im linken Bereich standardmäßig nur ein Abschnitt angezeigt wird, z. B. &quot;**[!UICONTROL Alle Projekte]**&quot;im Bereich &quot;**[!UICONTROL Projekte]**&quot;, muss ein &quot;[!DNL Workfront]&quot;-Administrator diesem Bereich beim Erstellen einer Layoutvorlage mindestens einen benutzerdefinierten Abschnitt hinzufügen und Sie dieser Vorlage zuweisen, bevor in diesem Bereich das linke Bedienfeld angezeigt wird.\
>Informationen dazu, wie ein [!DNL Workfront] -Administrator den linken Bereich in einer Layoutvorlage anpassen kann, finden Sie unter [Anpassen des linken Bereichs mithilfe einer Layoutvorlage](../../administration-and-setup/customize-workfront/use-layout-templates/customize-left-panel.md).

Der linke Bereich ist für die folgenden Bereiche verfügbar:

* [Standardabschnitte im Bereich [!UICONTROL Anforderungen]](#default-sections-in-the-requests-area)

## Verwenden des linken Navigationsbereichs

Das Suchen und Bearbeiten von Informationen im linken Bereich ist für diese Objekte ähnlich. Die im linken Bereich verfügbaren Optionen variieren je nach dem Objekt, auf das Sie zugreifen.

Informationen dazu, welche Abschnitte für bestimmte Bereiche verfügbar sind, finden Sie unter [[!UICONTROL Standardabschnitte] im linken Navigationsbereich](#default-sections-in-the-left-navigation-panel).

1. Klicken Sie auf das Symbol **[!UICONTROL Hauptmenü]** ![](assets/main-menu-icon-16x12.png) und dann auf den Namen eines Objekts, um darauf zuzugreifen.

   Die Objektseite wird angezeigt.

1. (Bedingt) Wenn Sie einen Bereich ausgewählt haben, der eine Objektliste enthält, z. B. **[!UICONTROL Projekte]** oder **[!UICONTROL Portfolios]**, müssen Sie Folgendes tun, um auf die Navigation im linken Bereich zuzugreifen:

   1. Klicken Sie auf ein Objekt in der Liste.
   1. (Optional) Im linken Fensternavigationsmenü können Sie ein anderes Objekt, z. B. **[!UICONTROL Aufgaben]** oder **[!UICONTROL Probleme]** innerhalb eines Projekts, im linken Bereich auswählen, um auf eine Liste von Objekten zuzugreifen. Klicken Sie dann auf den Namen des spezifischen Objekts.\

      Die Objektseite wird angezeigt.

1. Klicken Sie auf der Objektseite auf einen der Abschnitte im linken Bereich, um die Informationen in diesem Abschnitt anzuzeigen oder zu bearbeiten.

   Sie müssen über die Berechtigungen zum Bearbeiten des Objekts verfügen, bevor Sie die Informationen bearbeiten können.

   ![Abschnitte des linken Bedienfelds](assets/NWE-left-navigation-panel.png)

1. (Optional) Klicken Sie unten im linken Bedienfeld auf **[!UICONTROL Mehr anzeigen]** , um weitere Abschnitte anzuzeigen.

   >[!NOTE]
   >
   >Diese Option ist nicht verfügbar, wenn standardmäßig alle Abschnitte im linken Bereich angezeigt werden.

1. (Optional) Um einen Dashboard-Schnelllink hinzuzufügen, klicken Sie unten im linken Bereich auf **[!UICONTROL Dashboard hinzufügen]** und führen Sie dann die folgenden Schritte aus:

   1. Geben Sie im Feld **[!UICONTROL Name des Quick-Links]** einen Namen für das Dashboard ein.
   1. Geben Sie den Namen eines vorhandenen Dashboards in das Feld **[!UICONTROL Dashboard auswählen]** ein und klicken Sie dann auf das Dashboard, wenn es in der Liste angezeigt wird.

      >[!TIP]
      >
      >Sie müssen das Dashboard erstellen, bevor es in der Liste angezeigt wird.

   1. Klicken Sie auf **[!UICONTROL Hinzufügen]**.

      Der Dashboard-Link wird unten im linken Bereich angezeigt.

1. (Optional) Um die Reihenfolge eines Bereichs im linken Bereich zu ändern, klicken Sie auf das Symbol **[!UICONTROL 1} ziehen ![](assets/drag-icon.png) rechts von einem Abschnitt und ziehen Sie ihn nach oben oder unten.]**

   ![Ziehen eines Abschnitts](assets/drag-section-right.png)

1. (Optional) Klicken Sie auf das Symbol **[!UICONTROL Reduzieren]** ![](assets/left-panel-collapse-icon.png), um das linke Bedienfeld zu schließen.

   >[!NOTE]
   >
   >Wenn Sie das linke Bedienfeld reduzieren, behält [!DNL Workfront] Ihre Voreinstellung bei, selbst wenn Sie zu einem anderen [!DNL Workfront] -Objekt navigieren. Um das linke Bedienfeld erneut zu erweitern, klicken Sie auf das Symbol **[!UICONTROL Erweitern]** ![](assets/left-panel-expand-icon.png) .
