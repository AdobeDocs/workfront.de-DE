Sie können Informationen in einem benutzerdefinierten Formular bearbeiten, nachdem das Formular an ein Objekt angehängt wurde. ## Voraussetzungen

* Ihr Workfront-Administrator oder ein Planbenutzer mit Administratorzugriff auf benutzerdefinierte Formulare muss benutzerdefinierte Formulare in Ihrer Umgebung erstellen. Weitere Informationen finden Sie unter [Erstellen oder Bearbeiten eines benutzerdefinierten Formulars](../../administration-and-setup/customize-workfront/create-manage-custom-forms/create-or-edit-a-custom-form.md).
* Sie müssen benutzerdefinierte Formulare an ein Objekt anhängen.

  Informationen zum Anwenden benutzerdefinierter Formulare auf ein Objekt finden Sie unter [Hinzufügen eines benutzerdefinierten Formulars zu einem Objekt](../../workfront-basics/work-with-custom-forms/add-a-custom-form-to-an-object.md).

## Informationen in einem benutzerdefinierten Formular bearbeiten

Die Informationen in einem benutzerdefinierten Formular, das an ein Objekt angehängt wird, sind für alle Objekte identisch. Informationen dazu, welche Objekte ein benutzerdefiniertes Formular haben können, finden Sie unter [Übersicht über benutzerdefinierte Formulare](../../administration-and-setup/customize-workfront/create-manage-custom-forms/custom-forms-overview.md).

1. Navigieren Sie zu einem Objekt, für das Sie Informationen im benutzerdefinierten Formular bearbeiten möchten.
1. Klicken Sie im linken Bereich auf **`<Object type>`Details** .

   Klicken Sie beispielsweise beim Bearbeiten von Informationen zu einem benutzerdefinierten Projekt-Formular auf **Projektdetails**.

1. Scrollen Sie zum benutzerdefinierten Formular. Wenn ein benutzerdefiniertes Formular an das Objekt angehängt ist, wird der Name des Formulars im Abschnitt &quot;Details&quot;als Bereich angezeigt.
1. Klicken Sie bei Bedarf auf den Pfeil &quot;![](assets/expand-arrow-right.png)&quot; links neben dem Namen des benutzerdefinierten Formulars, um es zu erweitern.
1. Klicken Sie in der rechten oberen Ecke der Seite auf das Bearbeitungssymbol ![](assets/edit-icon.png).
1. Geben Sie Informationen in alle Felder ein, auf die Sie Zugriff haben.

   ![](assets/click-in-field-to-edit-info-350x132.png)

   Oder

   Wenn noch keine Informationen im Formular eingegeben wurden, klicken Sie für jedes Feld, auf das Sie Zugriff haben, auf &quot;**Hinzufügen+**&quot;, und beginnen Sie mit der Eingabe von Informationen.

   ![](assets/plus-add-to-edit-info-350x180.png)

   Wenn mehrere benutzerdefinierte Formulare an das Objekt angehängt sind, können Sie dies für jedes Formular tun.

   Beachten Sie je nach Feldtyp Folgendes:

   * Sie können nur eine Option für Optionsfelder auswählen.
   * Je nachdem, wie der Formularersteller das Feld konfiguriert hat, können Sie eine oder mehrere Optionen in einem Kontrollkästchen auswählen.
   * Je nachdem, wie der Formularersteller das Feld konfiguriert hat, können Sie eine oder mehrere Optionen in einem Dropdown-Feld mit Mehrfachauswahl auswählen.
   * Textfelder (fett, kursiv oder unterstrichen) können nur formatiert werden, wenn der Benutzer, der das Formular erstellt hat, sie als Textfeld mit Feldtyp &quot;Formatierung&quot;eingerichtet hat. Einzelzeilentextfelder und Absatztextfelder können nicht formatiert werden.
   * Sie können die Tageszeit in einem Datumsfeldtyp nur dann aktualisieren, wenn der Benutzer, der das Formular erstellt hat, es bei der Erstellung des Felds eingeschlossen hat.

   Weitere Informationen zu allen Feldtypen finden Sie unter [Erstellen oder Bearbeiten eines benutzerdefinierten Formulars](../../administration-and-setup/customize-workfront/create-manage-custom-forms/create-or-edit-a-custom-form.md).

1. Klicken Sie auf **Änderungen speichern**.

   >[!IMPORTANT]
   >
   >Sie müssen alle erforderlichen Felder im Formular ausfüllen, bevor Sie das Formular speichern können. Dem Namen eines erforderlichen Felds folgt ein Sternchen.
   >
   >![](assets/nwe-required-custom-field.png)

   Wenn jemand Daten in einem anderen Objekt ändert, auf das von berechneten benutzerdefinierten Feldern in Ihrem Objekt verwiesen wird, werden die Änderungen nicht automatisch in Ihrem Objekt übernommen. Informationen zum manuellen Aktualisieren aller berechneten benutzerdefinierten Felder in Ihrem Objekt finden Sie unter [Alle berechneten benutzerdefinierten Felder für ein Objekt neu berechnen](#recalculate-all-calculated-custom-fields-for-an-object) in diesem Artikel.

   Wenn abhängige Felder auf der Seite geändert werden, werden berechnete Felder im benutzerdefinierten Formular dynamisch in Echtzeit neu berechnet. Sie können den neuen berechneten Feldwert sehen, ohne das Formular zu speichern. Er wird jedoch erst dann auf das Formular und das Objekt angewendet, wenn Sie die Änderungen speichern. Dies gilt für berechnete Felder in Standardformularen sowie für benutzerdefinierte Formulare.

   Sie können auch alle berechneten benutzerdefinierten Felder für ein Objekt manuell aktualisieren, wenn Sie das Objekt zusammen mit anderen Objekten in einer Liste stapelweise bearbeiten. Anweisungen finden Sie unter [Neuberechnen aller berechneten benutzerdefinierten Felder für mehrere Objekte in einer Liste beim Bearbeiten der Objekte](#recalculate-all-calculated-custom-fields-for-multiple-objects-in-a-list-when-editing-the-objects) in diesem Artikel.

## Alle berechneten benutzerdefinierten Felder für ein Objekt neu berechnen  {#recalculate-all-calculated-custom-fields-for-an-object}

1. Rufen Sie die Hauptseite des Objekts auf, dessen benutzerdefinierte Felder Sie neu berechnen möchten.
1. Klicken Sie auf das Menü **Mehr** rechts neben dem Objektnamen und dann auf **Ausdrücke neu berechnen**.![](assets/more-icon.png)

   Dadurch werden alle benutzerdefinierten Felder im Formular des Objekts neu berechnet.

## Alle berechneten benutzerdefinierten Felder für mehrere Objekte in einer Liste beim Bearbeiten der Objekte neu berechnen {#recalculate-all-calculated-custom-fields-for-multiple-objects-in-a-list-when-editing-the-objects}

Sie können die benutzerdefinierten Felder mehrerer Objekte manuell neu berechnen, indem Sie sie stapelweise aus einer Liste oder einem Bericht bearbeiten.

1. Gehen Sie zu einer Liste von Objekten, die benutzerdefinierte Formulare mit berechneten Feldern enthalten.
1. Wählen Sie die Objekte aus, deren berechnete benutzerdefinierte Felder Sie aktualisieren möchten.
1. Klicken Sie auf das Symbol **Bearbeiten**.
1. Klicken Sie im linken Menü auf **Benutzerdefinierte Forms** und wählen Sie dann **Benutzerdefinierte Ausdrücke neu berechnen** aus.
1. Klicken Sie auf **Speichern** **Änderungen**.

   Workfront berechnet alle benutzerdefinierten Felder für alle ausgewählten Objekte.

>[!TIP]
>
>Je nach Komplexität Ihrer Projekte empfehlen wir, bei der Neuberechnung berechneter benutzerdefinierter Felder nicht viele Projekte auszuwählen, um eine optimale Leistung zu gewährleisten. Einige Dinge, die ein Projekt zu komplex machen könnten, sind möglicherweise mehrere Abhängigkeiten oder Zuweisungen oder eine große Anzahl benutzerdefinierter Felder.
>
>So berechnen Sie benutzerdefinierte Ausdrücke stapelweise aus einer Projektliste neu:
>
>1. Gehen Sie zu einer Projektliste oder einem Bericht und wählen Sie ein oder mehrere Projekte aus.
>1. Klicken Sie auf das Menü **Mehr** ![](assets/more-icon.png) und dann auf **Benutzerdefinierte Ausdrücke neu berechnen**.
>
>![](assets/recalculate-expressions-timeline-finances-drop-down-in-project-list-nwe.png)
>
>Workfront berechnet alle benutzerdefinierten Felder für alle ausgewählten Projekte.
