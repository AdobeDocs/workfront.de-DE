# Erste Schritte mit [!UICONTROL Home]

<!--Audited: 12/2023-->

In diesem Artikel wird ein Überblick über den aktuellen Bereich [!UICONTROL Home] in [!DNL Adobe Workfront] beschrieben. Weitere Informationen zum Arbeiten mit der neuen [!UICONTROL Startseite] finden Sie unter [Erste Schritte mit der neuen Startseite](../new-home/get-started-with-new-home.md).

Es gibt zwei Möglichkeiten, Ihre Arbeit im aktuellen Bereich [!UICONTROL Home] anzuzeigen:

* Aus der [!UICONTROL Arbeitsliste]
* Vom [!UICONTROL Kalender] aus.

## [!UICONTROL Arbeitsliste]

Sie können alle zugewiesenen Arbeiten an einem Ort in der [!UICONTROL Arbeitsliste] anzeigen:

![[!UICONTROL Arbeitsliste] Bedienfelder](assets/worklist-and-right-panel-home.png)

Überprüfen Sie die folgenden Abschnitte im aktuellen Bereich [!UICONTROL Home] , wie im obigen Bild hervorgehoben:

* **A**: Verwenden Sie das linke Bedienfeld, um zugewiesene Arbeiten zu filtern, zu sortieren und auszuwählen.

* **B**: Nachdem Sie ein Arbeitselement im linken Bereich ausgewählt haben, verwenden Sie das rechte Bedienfeld, um mit benutzerdefinierten Feldern zu interagieren, Aktualisierungen vorzunehmen und die Protokollzeit zu ändern.

Weitere Informationen zur Verwendung der [!UICONTROL Hausarbeitsliste] finden Sie in den folgenden Artikeln:

* [Elemente in der [!UICONTROL Arbeitsliste] im Bereich [!UICONTROL Startseite] anzeigen](../../../workfront-basics/using-home/using-the-home-area/display-items-in-home-work-list.md)
* [Arbeitselemente aus dem Bereich [!UICONTROL Home] erstellen](../../../workfront-basics/using-home/using-the-home-area/create-work-items-in-home.md)

## [!UICONTROL Kalender]

Sie können Ihre Arbeit visualisieren, indem Sie anzeigen, wann Arbeitselemente fällig sind, und Sie können die Zeit für ihre Fertigstellung blockieren, indem Sie Ihre Arbeit im [!UICONTROL Kalender] im Bereich [!UICONTROL Startseite] überprüfen:

![](assets/calendar-home-nwe-350x114.png)

Überprüfen Sie die folgenden Abschnitte im aktuellen Bereich [!UICONTROL Home] , wie im obigen Bild hervorgehoben:

* **A**: Verwenden Sie die Ansicht [!UICONTROL Kalender] , um eine visuelle Karte der Arbeit zu erstellen, die Sie abschließen müssen. Klicken Sie einfach auf und ziehen Sie Arbeitselemente aus der [!UICONTROL Arbeitsliste] in den [!UICONTROL Kalender], um die Arbeitszeit zu blockieren.

* **B**: Verwenden Sie die Synchronisierungsoption von [!UICONTROL Kalender], um eine Synchronisierung mit Ihrem [!DNL Outlook] -Kalender durchzuführen. Verwalten Sie Meetings und arbeiten Sie alle an einem Ort.

* **C**: Mit der Schaltfläche [!UICONTROL Details] können Sie einen neuen rechten Bereich öffnen, in dem Sie weitere Informationen zum Arbeitselement anzeigen können.

* **D**: Verwenden Sie die Symbolleiste, um zu verfolgen, wann Ihnen zugewiesene Arbeit fällig ist.

Weitere Informationen zur Verwendung des Home-Kalenders finden Sie unter [Verwenden der Ansicht [!UICONTROL Home Calendar]](../../../workfront-basics/using-home/using-the-home-area/use-home-calendar-view.md).

## Nach Ihrer Arbeit suchen

Der Bereich [!UICONTROL Home] ist Ihre zentrale Anlaufstelle, um Ihnen zugewiesene Aufgaben, Probleme und Genehmigungen zu finden. Die Filter-, Gruppierungs- und Sortieroptionen arbeiten zusammen, um Sie bei der gewünschten Organisation Ihrer Arbeitselemente zu unterstützen. Sie können Gruppierungen verwenden, um Elemente zu gruppieren, Filter zu verwenden, die sich auf zu vervollständigende Arbeiten konzentrieren, und schließlich eine Sortierung, um die Elemente in auf- oder absteigender Reihenfolge Ihrer Gruppierung aufzulisten.

>[!NOTE]
>
>Die Filter- und Sortieroptionen werden im Browser gespeichert. Wenn Sie durchgängig denselben Browser auf demselben Computer verwenden (und die Site-Daten nicht löschen), ändern sich die Filter und die Sortierung nicht, aber wenn Sie Browser oder Computer wechseln, werden die Filter und die Sortierung unterschiedlich sein.

### Gruppieren ähnlicher Arbeitselemente zusammen mit Gruppierungen

Sie können Gruppierungen verwenden, um in der Arbeitsliste unter einer gemeinsamen Kopfzeile wie Elemente anzuzeigen. Sie können Elemente gruppieren, indem Sie:

* [!UICONTROL Geplanter Abschluss]
* [!UICONTROL Geplanter Start]
* [!UICONTROL Datum der Übermittlung]
* [!UICONTROL Projekt]
* [!UICONTROL Meine Priorität]

Weitere Informationen zur Verwendung von Gruppierungen finden Sie unter [Anzeigen von Elementen in der [!UICONTROL Arbeitsliste] im Bereich [!UICONTROL Startseite]](../../../workfront-basics/using-home/using-the-home-area/display-items-in-home-work-list.md).

![](assets/group-by-drop-down-expanded-in-home-with-planned-start-date-nwe-350x273.png)

### Fokus mit Filtern eingrenzen

Mit dem Filter [!UICONTROL Arbeitsliste] können Sie Ihren Fokus auf bestimmte Arbeitselemente beschränken.

Die folgenden Beispiele zeigen, wie Sie Elemente filtern können, die Sie auf der Startseite anzeigen:

* Filtern nach Elementtyp und -status.

  Wenn Sie beispielsweise alle Aufgaben anzeigen möchten, können Sie den Filter [!UICONTROL Aufgaben] auswählen. Wenn Sie genauer sein möchten und nur Aufgaben anzeigen möchten, die für die Arbeit bereit sind, wählen Sie unter dem Filter [!UICONTROL Aufgaben] die Filter [!UICONTROL Bereit zum Start] aus.

* Filtern Sie nur nach Elementtyp.

  Sie können beispielsweise [!UICONTROL Probleme] auswählen, um alle Probleme in allen Status ([!UICONTROL Arbeiten an] oder [!UICONTROL angefordert]) oder [!UICONTROL Genehmigungen] anzuzeigen, um alle Arbeitselemente, Zugriffsanfragen, Zeitpläne, Dokumente und Testversandgenehmigungen anzuzeigen.

Weitere Informationen zur Verwendung von Filtern finden Sie unter [Anzeigen von Elementen in der [!UICONTROL Arbeitsliste] im Bereich [!UICONTROL Startseite]](../../../workfront-basics/using-home/using-the-home-area/display-items-in-home-work-list.md).

![](assets/displaying-work-items-filters-nwe-350x401.png)

### Für Ihr Team zugewiesene Zugriffsaufgaben

Der Bereich [!UICONTROL Home] verfügt über eine permanente Gruppierung ([!UICONTROL Team Requests]), die Aufgaben und Probleme anzeigt, die Ihren Teams zugewiesen sind. In diesem Abschnitt werden Teams angezeigt. Wenn Sie auf den Namen eines Teams klicken, wird die Seite Team-Anforderungen dieses Teams geöffnet.

>[!NOTE]
>
>Die Filter- und Sortieroptionen wirken sich nicht auf die Gruppierung [!UICONTROL Team Requests] aus. Diese Gruppierung ist sichtbar, solange Sie Ihrer Mannschaft Arbeit zugewiesen haben.

Weitere Informationen zum Zugriff auf Teamanfragen finden Sie im Artikel [[!UICONTROL Verwalten] von Arbeits- und Teamanfragen im Bereich [!UICONTROL Startseite]](../../../workfront-basics/using-home/using-the-home-area/manage-work-and-team-requests-home.md).

![](assets/team-requests-expanded-home-group-by-drop-down-nwe-350x314.png)

### Überwachen Sie die von Ihnen gesendeten Arbeiten.

Verfolgen Sie die Arbeit, die Sie zur Genehmigung vorgelegt haben, direkt über die [!UICONTROL Arbeitsliste]. Es gibt eine permanente Gruppierung für [!UICONTROL Von mir gesendete Genehmigungen], wenn Sie in der [!UICONTROL Arbeitsliste] die Ansicht von [!UICONTROL Genehmigungen] auswählen. Sie können den Genehmiger an die Arbeit erinnern, die genehmigt werden muss. Sie können die Validierung bei Bedarf auch zurückrufen.

>[!NOTE]
>
>Der Filter und die Sortierung wirken sich nicht auf die Gruppierung [!UICONTROL Validierungen, die ich gesendet habe] aus. Diese Gruppierung ist sichtbar, solange die Arbeit auf die Validierung wartet.
>
>Die Anzahl der Genehmigungen im Filter [!UICONTROL Genehmigungen] zeigt nicht die Anzahl der Elemente an, die Sie zur Genehmigung eingereicht haben. Der Filter [!UICONTROL Genehmigungen] zeigt nur die Elemente an, die auf Ihre Genehmigung warten.

![](assets/approvals-expanded-home-group-by-drop-down-nwe-350x415.png)

## Was für Sie wichtig ist, priorisieren

Mit der [!UICONTROL Arbeitsliste] können Sie Arbeiten aufdecken und priorisieren, die für Sie mit der Gruppierung [!UICONTROL Meine Priorität] wichtig sind. Sie können Ihrer Liste [!UICONTROL Meine Priorität] bis zu 20 Elemente hinzufügen.

>[!NOTE]
>
>Alle Arbeitselemente, die [!UICONTROL Meine Priorität] hinzugefügt wurden, sind nur für Sie sichtbar, und niemand kann die Arbeit für Sie priorisieren.

Weitere Informationen zur Verwendung von [!UICONTROL My Priority] finden Sie unter [Priorize work in the [!UICONTROL Home] area](../../../workfront-basics/using-home/using-the-home-area/prioritize-work-in-home.md).

![](assets/prioritizing-in-home-group-by-drop-down-nwe-350x412.png)

## Erledigen

Nachdem Sie auf den Namen eines Elements in der [!UICONTROL Arbeitsliste] geklickt haben, verwenden Sie das rechte Bedienfeld in [!UICONTROL Startseite], um Ihre Arbeit zu aktualisieren und abzuschließen.

### Arbeiten in einem für Sie angepassten Raum

Ihr [!DNL Workfront] -Administrator kann bis zu 16 benutzerdefinierte Felder zu Elementen in der [!UICONTROL Arbeitsliste] hinzufügen, um die Suche und Aktualisierung der benötigten Informationen zu vereinfachen. Die meisten Felder können bearbeitet werden, sodass Sie nicht zum Arbeitselement navigieren müssen.

Um die Felder in diesem Bereich zu aktualisieren, klicken Sie einfach auf ein Feld. Wenn das Feld bearbeitbar ist, können Sie es im rechten Bereich ändern.

>[!NOTE]
>
>Sie können das [!UICONTROL Veröffentlichungsdatum] nicht aktualisieren, wenn dieses Feld einem Arbeitselement aus einer Layoutvorlage hinzugefügt wird.

![](assets/16-fields-home-350x137.png)

### Aktualisieren anderer Personen mit wichtigen Informationen

Kommunizieren Sie schnell und effizient mit anderen aus dem Bereich [!UICONTROL Home] . Im Kommentarbereich eines Arbeitselements können Sie Fragen stellen, andere über aktuelle Updates benachrichtigen oder auf direkte Kommentare aus dem Bereich [!UICONTROL Startseite] antworten. Darüber hinaus können Sie andere über den Fortschritt informieren, indem Sie:

* Aktualisieren des [!UICONTROL Veröffentlichungsdatums] im Kommentarbereich

  >[!NOTE]
  >
  >   Dies ist der einzige Ort, an dem Sie das Datum der Veröffentlichung auf der Startseite aktualisieren können.

* Berichte dazu, wie Sie fühlen, dass Dinge vor sich gehen, durch Hinzufügen eines Kommentars oder Aktualisieren der [!UICONTROL Bedingung] eines Elements
* Aktualisieren der prozentualen Abschlussleiste

Weitere Informationen zur Kommunikation im Bereich [!UICONTROL Home] finden Sie unter [Update work](../../../workfront-basics/updating-work-items-and-viewing-updates/update-work.md).

![](assets/updates-home-350x198.png)

### Zeit protokollieren

Verfolgen Sie die Stunden, die Sie an einzelnen Arbeitselementen gearbeitet haben. Sie können einen allgemeinen Stundentyp auswählen, z. B. [!UICONTROL Task Time], oder Sie können einen projektspezifischen Stundentyp auswählen, der von Ihrem [!DNL Workfront] -Administrator eingerichtet wurde.

Weitere Informationen zur Protokollzeit im Bereich [!UICONTROL Home] finden Sie im Abschnitt [Home](../../../timesheets/create-and-manage-timesheets/log-time.md#home) im Artikel [Log time](../../../timesheets/create-and-manage-timesheets/log-time.md).

![](assets/log-time-home-350x181.png)
