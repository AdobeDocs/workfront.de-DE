# Entfernen von Berichten aus einem Dashboard

>[!IMPORTANT]
>
>Die Funktion Canvas-Dashboards ist derzeit nur für Benutzer verfügbar, die an der Beta-Phase teilnehmen. Weitere Informationen finden Sie unter [Beta-Informationen zu Canvas-Dashboards](/help/quicksilver/product-announcements/betas/canvas-dashboards-beta/canvas-dashboards-beta-information.md).

Sobald ein Dashboard erstellt wurde und Sie ihm Berichte hinzugefügt haben, können Sie ältere Berichte entfernen, die für dieses bestimmte Dashboard nicht mehr anwendbar sind.

Das Entfernen eines Berichts aus einem Dashboard ist dauerhaft. Wenn Sie einen Bericht nach dem Löschen erneut hinzufügen müssen, müssen Sie den Bericht neu erstellen.

## Voraussetzungen

Berichte müssen einem Dashboard hinzugefügt werden, bevor sie entfernt werden können.

## Entfernen eines Berichts aus einem Dashboard

>[!WARNING]
>
>Nachdem ein Bericht gelöscht wurde, kann er nicht mehr wiederhergestellt werden.

{{step1-to-dashboards}}

1. Klicken Sie im linken Bedienfeld auf **Arbeitsflächen-Dashboards**.

1. Wählen Sie auf **Seite** Dashboards“ das Symbol **Mehr** ![Mehr](assets/more-icon.png) in der oberen rechten Ecke des Dashboards aus, das Sie aktualisieren möchten, und klicken Sie dann auf **Entfernen**.

   ![Bericht entfernen](assets/delete-a-report.png)

1. Klicken Sie im **Bericht löschen** angezeigten Dialogfeld auf **Löschen**. Der Bericht wird gelöscht und aus dem Dashboard entfernt.
