# Erste Schritte mit der eigenständigen Adobe-Planung

In diesem Artikel wird beschrieben, wie Sie von der Verwendung der eigenständigen Version von Adobe Workfront Planning profitieren können.

## Wie Ihr Unternehmen eine Adobe Workfront Standalone Planning-Lizenz erwerben kann

Unabhängig davon, ob Sie Adobe Workfront-Kunde sind oder nicht, können Sie mit Ihrer Adobe-Kundenbetreuung über eine Workfront Planning Standalone-Lizenz sprechen. <!--do people HAVE to be WF or Adobe customers at all, or can anyone buy this?-->

## Integrierte und eigenständige Workfront-Planungsversionen

Mit der integrierten Version von Workfront Planning können Sie:

* Verbinden von Planning-Objekten mit Workfront-Objekten
* Hinzufügen benutzerdefinierter Felder aus Workfront zu Planungs-Datensatztypen
* Automatisches Erstellen von Workfront-Projekten basierend auf Planungsinformationen
* Siehe Planungsinformationen im Berichtssystem für das Arbeitsflächen-Dashboard von Workfront

Mit der eigenständigen Version von Workfront Planning können Sie:

* Verwenden Sie die Workfront Planning Standalone Open API, um Planning-Objekte in alle anderen Anwendungen zu integrieren <!--ask Lilit and I think this is an Integrated benefit also-->
