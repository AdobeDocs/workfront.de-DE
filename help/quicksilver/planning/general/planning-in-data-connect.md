# Workfront Data Connect für Workfront Planning

Workfront Data Connect bietet mithilfe von Snowflake-Technologie Zugriff auf Workfront-Daten, einschließlich aus Adobe Workfront Planning, sodass Sie geschäftskritische Fragen beantworten und den Wert Ihrer Prozesse messen können. Unternehmenskunden können ihre Daten mithilfe von Datenvisualisierungs-Tools direkt aus einem Data Lake von Workfront analysieren oder Workfront-Daten in kundeneigene Data Warehouses exportieren.

Abonnenten können ihr Business-Intelligence-Tool ihrer Wahl (Tableau, Power BI, Domo, Looker usw.) Verbinden von Daten mit , um Datenvisualisierungen mit den folgenden Datentypen zu erstellen:

* Niedrige Latenz, aktuelle Statusdaten
* Ändern von Ereignisdatensätzen für Point-in-Time-Vergleiche
* Tägliche historische Momentaufnahmen zur Trendanalyse

Informationen zum Erstellen eines Leserkontos mit Snowflake und zum Herstellen einer Verbindung mit Workfront Data Connect finden Sie in den folgenden Artikeln:

* [Erstellen eines Leserkontos oder einer Verbindung für Snowflake](/help/quicksilver/reports-and-dashboards/data-lake/create-a-reader-account.md)
* [Herstellen einer Verbindung mit Workfront Data Connect](/help/quicksilver/reports-and-dashboards/data-lake/share-data-externally.md)


Weitere Informationen zum Planen im Datenwörterbuch für Workfront Data Connect finden Sie unter [Datenwörterbuch für Workfront Data Connect](/help/quicksilver/reports-and-dashboards/data-lake/data-dictionary.md).
